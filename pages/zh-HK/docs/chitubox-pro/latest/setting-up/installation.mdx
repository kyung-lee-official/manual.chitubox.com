# 安裝

## Windows

雙擊 CHITUBOX 安裝包開始安裝，選擇安裝流程語言為繁體中文。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation001.png"
	}
/>

點擊 **下一步**。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation002.png"
	}
/>

接受 EULA。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation003.png"
	}
/>

選擇你的安裝路徑，點擊 **安裝** 開始安裝。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation004.png"
	}
/>

等待安裝。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation005.png"
	}
/>

安裝完成後，點擊**是**，資源管理器將會重啟。

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation006.png"
	}
/>

點擊 **完成**，完成安裝！

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-installation007.png"
	}
/>

<Admonition type="note" title="注意">
	如果更新後的版本仍然無法解決更新日誌中提到的問題，可能是因為緩存未清理。請點擊
	<NextLink href="/zh-CN/docs/faq/latest/chitubox-basic-faq#清除缓存">此處</NextLink>
	查看如何清理緩存。
</Admonition>
