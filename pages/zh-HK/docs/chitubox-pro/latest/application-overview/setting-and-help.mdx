<StyledAntdImage

-   自動保存間隔

    自動保存間隔：自動保存的時間間隔。CHITUBOX 會在**自動保存目錄**（默認為日誌路徑）下生成一個臨時的 _.chitubox_ 文件。如果 CHITUBOX 正常退出，則該臨時文件會被刪除。

-   自動保存目錄

    您可以手動輸入需要保存的目錄，也可以點擊 <button className="button button--secondary">...</button> 選擇文件夾作爲自動保存目錄。

-   檔案名格式

    格式應以 `%{f}_` 開頭，後面跟上 `%{date_format}`。

-   重讀名稱

    格式應以 `%{f}_` 開頭，後面跟上 `%{nnn:0:1}`。

    其中： <br />
    `nnn` 代表編號寬度，例如 `000`。 <br />
    `0` 代表起始值。<br />
    `1` 代表步長。
