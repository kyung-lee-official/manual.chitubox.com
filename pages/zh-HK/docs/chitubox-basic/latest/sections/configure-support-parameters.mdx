# 设置支撑参数

点击主窗口右上方的支撑图标 <ChituboxSupport />进入支撑配置界面。

## Z 抬升高度

模型最低点与打印平台之间的高度距离。默认值为 5 mm。

## 支撑设置

CHBITUBOX 提供了 3 种默认的支撑设置：细、中、粗。如果它们不适合您的模型，您可以自定义支撑设置并导出以供将来使用。您可以在此处 <ChituboxProfileImportIcon /> 导入、<ChituboxProfileExportIcon /> 导出和 <ChituboxProfileResetIcon /> 重置支撑设置。

## 自动支撑

自动支撑有两种模式：**+全部** 和 **+平台**。

**+全部** 允许支撑在模型表面生成，而 **+平台** 只允许支撑在平台上生成。

## 操作模式

有三种操作模式：**添加**、**删除** 和 **编辑**。

### 添加

<ChituboxAddSupportIcon /> 点击模型添加支撑。

### 删除

<ChituboxDeleteSupportIcon /> 点击选择想要删除的支撑，所选支撑将变为红色，再次点击按钮或按下
Delete 键删除支撑。

### 编辑

<ChituboxEditSupportIcon /> 点击选择想要编辑的支撑，所选支撑将变为绿色，按住
Ctrl 键可以选择多个支撑，可以调整选中支撑的参数，也可以拖动支撑调整支撑位置。

## 清除全部

点击此按钮，所有支撑都会被清除。
