# 風險檢測

## 空腔檢測

空腔檢測位於主面板右側，文件列表面板下方。

空腔檢測可以幫助用戶識別 3D 列印模型中的任何封閉空腔，例如封閉的口袋或空洞，並且用戶可以向模型添加孔以避免真空效應。

## 碰撞檢測

碰撞檢測位於主面板右側，文件列表面板下方。

碰撞檢測可以幫助用戶突出顯示 3D 列印模型中的碰撞，用戶可以調整模型以避免碰撞。

## 懸空檢測

懸空檢測位於支撐視圖中，點擊 <ChituboxSupport /> 切換到支撐視圖。懸空檢測位於支撐視圖右側，支撐視圖位於支撐列表面板下方。

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/zh-CN/chitubox-basic/2.x.x/008-risk-detection-suspend-detection.png"
		}
	/>
</ImageContainer>

懸空檢測功能可以幫助用戶識別 3D 列印模型中任何未加支撐的高風險區域，並且用戶可以向模型添加支撐以避免列印失敗。

它提供三種精度級別：低、中和高。精度級別越高，檢測結果越準確，但檢測時間越長。通過點擊“刷新”按鈕 <ChituboxRefreshIcon />，您可以刷新檢測結果。

點擊“懸空區域”面板中的箭頭會自動將模型中的相應區域放大。

## 孤島檢測

孤島檢測與懸空檢測類似，但它是通過迭代 2D 圖層而不是 3D 模型的方式來檢測孤島。要使用孤島檢測，需要點擊主面板右側、文件列表面板下方的 **切片** 按鈕對模型進行切片。

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/zh-CN/chitubox-basic/2.x.x/008-risk-detection-slice.png"
		}
	/>
</ImageContainer>

切片模型後，您會進入預覽窗口，並在預覽窗口頂部看到 **檢測孤島** 和 **刪除孤島** 按鈕。

<ChituboxIslandsDetectionIcon /> 檢測孤島將檢測模型中的孤島，並在預覽窗口中以紅色突出顯示已檢測到的孤島。

<ChituboxDeleteIslandsIcon /> 刪除孤島將刪除已檢測到的孤島。通常，您不希望直接使用此功能，而是應該返回並為已檢測到的孤島添加支撐。僅當檢測到的孤島僅存在於少數連續的幾個圖層中，並且您想要刪除它們時才使用此功能。
