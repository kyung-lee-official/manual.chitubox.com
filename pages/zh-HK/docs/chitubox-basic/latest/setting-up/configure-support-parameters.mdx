## 支撑設定

你可以分别編輯支撑的<b>頂部</b>，<b>中部</b>，<b>底部</b>以及<b>底筏</b>。

### 頂部

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.top.png"
	}
/>

| 參數                 | 説明                             |
| -------------------- | -------------------------------- |
| 接觸形狀             | 支撐頂部與模型接觸形狀的選擇     |
| 接觸形狀直徑（毫米） | 支撐頂部與模型接觸部位的直徑尺寸 |
| 接觸深度（毫米）     | 支撐頂部與模型接觸的深度尺寸     |
| 連接形狀             | 頂部支撐主体的連接形狀           |
| 上端直徑（毫米）     | 頂部支撐主体上端的直徑尺寸       |
| 下端直徑（毫米）     | 頂部支撐主体下端的直徑尺寸       |
| 連接長度（毫米）     | 頂部支撐主体的長度尺寸           |

### 中部

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.mid.png"
	}
/>

| 參數                 | 説明                             |
| -------------------- | -------------------------------- |
| 形狀                 | 中部支撐主体的形狀選擇           |
| 直徑（毫米）         | 支撐主体的直徑尺寸               |
| 角度 (°)             | 中部支撐與頂部支撐的最小夾角     |
| 小柱子形狀           | 小柱支撐主体的形狀               |
| 直徑（毫米）         | 小柱支撐主体的直徑尺寸           |
| 上端接觸深度（毫米） | 小柱支撐上端與模型的接觸深度尺寸 |
| 下端接觸深度（毫米） | 小柱支撐下端與模型的接觸深度尺寸 |

### 底部

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.bot.png"
	}
/>

| 參數             | 説明                                                     |
| ---------------- | -------------------------------------------------------- |
| 平台接觸形狀     | 選擇支撐與打印平台的接觸形狀                             |
| 接觸直徑（毫米） | 與打印平台接觸的支撐部位的直徑尺寸                       |
| 厚度             | 與打印平台接觸的支撐部位的厚度尺寸                       |
| 模型接觸形狀     | 當支撐底部落於模型上時，支撐下端部位與模型的接觸形狀     |
| 接觸直徑（毫米） | 當支撐底部落於模型上時，支撐下端部位與模型接觸的直徑尺寸 |
| 接觸深度（毫米） | 當支撐底部落於模型上時，支撐下端部位與模型接觸的深度尺寸 |
| 接觸點           | 當支撐底部落於模型上時，支撐下端部位與模型接觸點的數量   |

### 底筏

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.raf.png"
	}
/>

| 參數             | 説明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏形狀         | 選擇與打印平台接觸底筏的形狀                                             |
| 底筏面積比例 (%) | 設定底筏的面積比例，該值為底筏總面積相對於模型向打印平台投影總面積的比例 |
| 底筏厚度（毫米） | 底筏的厚度尺寸                                                           |
| 底筏高度（毫米） | 底筏的高度尺寸                                                           |
| 底筏坡度 (°)     | 底筏側面與打印平台的傾斜角度                                             |

## 自動/手動支撐

<p>
	<b>交叉寬度（毫米）</b>：手動添加支撐時，相鄰支撐是否添加交叉結構的最大距離值，當相鄰支撐距離超過該值，則不添加交叉結構。
</p>

<p>
