# 管理你的账号和许可

## 管理你的账号和许可

<Admonition type="danger" title="危险">
	请勿将你的许可序列号告诉任何人，获取许可序列号后应尽快绑定账号。
</Admonition>

<Admonition type="note" title="注意">
	确保你的电脑已经连接网络。
</Admonition>

1. 当 CHITUBOX 第一次启动时，登录窗口会弹出。

    <ImageContainer>
    	<StyledAntdImage
    		src={"/images/docs/zh-CN/chitubox-pro/004-setting-up-login002.png"}
    	/>
    </ImageContainer>

    如果你还没有账号，点击 <SpanA href="https://cc.chitubox.com/register">注册</SpanA> 进行注册。

    <ImageContainer>
    	<StyledAntdImage
    		src={
    			"/images/docs/zh-CN/chitubox-pro/004-setting-up-activate-license001.png"
    		}
    	/>
    </ImageContainer>

    <Admonition type="note" title="注意">
    	登录完成后将对账号进行检测，若账号不符合试用权限，则 <b>开始试用</b>{" "}
    	将不予显示。
    </Admonition>

1. 输入你的许可序列号：

    <ImageContainer>
    	<StyledAntdImage
    		src={
    			"/images/docs/zh-CN/chitubox-pro/004-setting-up-activate-license002.png"
    		}
    	/>
    </ImageContainer>

    点击**下一步**，将激活许可序列号并且将其绑定到你的电脑。

    <ImageContainer>
    	<StyledAntdImage
    		src={
    			"/images/docs/zh-CN/chitubox-pro/004-setting-up-activate-license003.png"
    		}
    	/>
    </ImageContainer>

    激活许可证要是和 3D 打印设备关联，则还需要输入打印机设备 ID:

    <ImageContainer>
    	<StyledAntdImage
    		src={
    			"/images/docs/zh-CN/chitubox-pro/004-setting-up-activate-license004.png"
    		}
    	/>
    </ImageContainer>

    如果你在 CHITUBOX 官网<SpanA href="https://cc.chitubox.com/login">登录</SpanA>你的账号，在 **我的产品** > **我的许可证** 下面应该可以看到你刚刚绑定的许可。

    <ImageContainer>
    	<StyledAntdImage
    		src={
    			"/images/docs/zh-CN/chitubox-pro/004-setting-up-add-your-license-to-your-account-005.png"
    		}
    	/>
    </ImageContainer>

    如果你已经用试用许可进入了 CHITUBOX Pro，点击<NextLink href="/zh-CN/docs/chitubox-pro/latest/application-overview/setting-and-help#授权">这里</NextLink>查看在哪里更新你的许可。

## 将 CHITUBOX 迁移到另一台电脑

如果你的电脑意外损坏了，或者你想要将 CHITUBOX 迁移到你的新电脑，你需要先将许可从旧电脑上解绑。

在网页端<SpanA href="https://cc.chitubox.com/login">登录</SpanA>你的 CHITUBOX 账号。

点击 **我的产品** > **我的许可证**

<ImageContainer>
	<StyledAntdImage
		src={
			"/images/docs/zh-CN/chitubox-pro/004-setting-up-add-your-license-to-your-account-006.png"
		}
	/>
</ImageContainer>

点击 **已绑 PC 管理** > **解绑**

<ImageContainer>
	<StyledAntdImage
		src={
			"/images/docs/zh-CN/chitubox-pro/004-setting-up-add-your-license-to-your-account-007.png"
		}
	/>
</ImageContainer>

你的许可将从旧电脑上解绑，然后你就可以重新在你的新电脑上激活你的许可。
