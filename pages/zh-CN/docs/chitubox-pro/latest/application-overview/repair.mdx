# 修复

## 修复面板

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-repair020.png"
	}
/>

修复面板上的下拉菜单可以选择**按模型显示**或者**按错误类型显示**，检测的错误类型包括**多余壳体**，**孔洞**，**坏边**，**反向**，**相交**。
如果选中**检测**选项，软件将会自动检测模型，并展示出模型的错误数据。基本修复可以完成绝大多数的修复任务，高级修复是针对清除多余壳体优化的修复。

## 左侧工具栏

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-repair023.png"
	}
/>

在修复模型时，如需对模型局部或者指定区域进行修复，我们可以通过**左侧工具栏**来快捷选取。快捷选择工具包括**线**、**笔刷**、**矩形**、**多边形**和**自由曲线**。
