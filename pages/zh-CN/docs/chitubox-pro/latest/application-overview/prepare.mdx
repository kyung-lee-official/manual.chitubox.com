## 镂空

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare008.png"
	}
/>

将选定的模型按照设定的镂空参数镂空。

<AntdImage
	src={"/images/docs/zh-CN/chitubox-pro/1.x.x/gif/003hollow.gif"}
	style={{ maxWidth: 200 }}
/>

<br />
<br />

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare008.1.png"
	}
/>

-   壁厚：镂空时被操作对象剩余的腔体厚度（**外镂空** 模式下时为被削去的厚度）。
-   模式：
    -   _内镂空_：镂空后剩余一个封闭腔体。
    -   _外镂空_：削去对对象外层，厚度为**壁厚**。
    -   _底部贯穿_：和*内镂空*类似，但不保留底部。
-   填充：开启/关闭生成内部填充结构。
-   精度：被镂空面生成的三角面数量。
-   动画：镂空时展示镂空动画（实际镂空进度以底部进度条为准）。
-   仅保留主腔：在模型镂空时只生成一个主腔体。

## 挖洞

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare009.png"
	}
/>

将选定的模型按照设定的挖洞参数挖洞。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare021.png"
	}
/>

-   挖洞模式：有贯穿和非贯穿模式两种。
-   多边形边数：边数越大，孔的形状将越接近圆形。
-   Φ：孔的直径。
-   B：孔能够穿透的最大深度。
-   C：孔与孔底部形成的夹角。
-   旋转：孔的旋转角度。
-   保留孔：开启/关闭保留打孔塞子。
-   角度：以垂直于模型/相机透视角度进行挖洞。
-   强制挖洞：开启/关闭强制挖洞。
-   限制深度 B：开启/关闭限制挖洞深度 B。

<Admonition type="note" title="注意">
	在 CHITUBOX Pro V1.3.0
	版本中优化了挖洞功能的算法，**取消了壁厚参数B**，软件将自动计算出壁厚并进行挖洞。
</Admonition>
