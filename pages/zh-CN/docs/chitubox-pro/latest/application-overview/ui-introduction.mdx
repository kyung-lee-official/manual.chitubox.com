<p>
	<b>移动</b>：将所选的模型移动到设定的位置。
</p>

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-ui-introduction-move.png"
	}
/>

-   放在底板上：以模型的最低点为基准点，将所选模型放在底板上。

-   居中：将所选模型居中放置在打印平台上。

-   重置：将所选模型的摆放位置重置。

<p>
	<b>旋转</b>：根据设定的旋转方向和角度旋转所选模型。
</p>

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-ui-introduction-rotate.png"
	}
/>

-   按面放平：以在模型上所选的面为基准面，将模型按基准面贴合打印平台放平。

-   组合中心：以所选的多个模型的组合的中心为旋转中心。

<p>
	<b>缩放</b>：根据设定的尺寸或缩放比例缩放所选模型。
</p>

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-ui-introduction-scale.png"
	}
/>

-   锁定比例：锁定所选模型的 X 轴，Y 轴和 Z 轴的比例进行缩放。

-   支撑直径不变：在保持支撑直径不变的情况下实现和模型同比例缩放。

-   缩放以适应：对所选模型进行缩放以适应打印平台。

-   重置：重置模型大小。

<p>
	<b>镜像</b>：对所选模型进行镜像操作。
</p>

<StyledAntdImage
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-ui-introduction-mirror.png"
	}
/>

-   保留原始模型：如果启用此选项，镜像后将保留原始模型。
