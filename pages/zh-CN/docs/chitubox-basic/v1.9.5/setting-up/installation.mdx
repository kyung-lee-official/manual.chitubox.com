# 安装

## Windows

双击 CHITUBOX 安装包开始安装，选择安装流程语言为简体中文。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation001.png"
	}
/>

点击 **下一步**。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation002--1.png"
	}
/>

接受 EULA。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation003--1.png"
	}
/>

选择你的安装路径，点击 **安装** 开始安装。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation004--1.png"
	}
/>

等待安装。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation005--1.png"
	}
/>

点击 **是**，资源管理器将会重启。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation006--1.png"
	}
/>

点击 **完成**，完成安装！

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-installation007--1.png"
	}
/>
