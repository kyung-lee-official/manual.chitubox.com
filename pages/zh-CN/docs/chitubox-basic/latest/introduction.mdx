# 介绍

CHITUBOX Basic 是一款免费的 3D 打印切片软件，可将 3D 网格文件切片并转换为可打印文件。它旨在让 3D 打印更轻松、更高效，简化整个 3D 打印过程，让用户能够简单快速地将 3D 模型转换为 3D 打印件。软件支持多种 3D 打印机和配置文件，并且提供了丰富的选项来优化 3D 打印模型。

## 新特性 ✨

### 全新的 UI 界面

<ThemeScrollSlider lang={"en"} />

很高兴地宣布 CHITUBOX Basic 2 推出了全新的用户界面。新的 UI 界面让用户体验更轻松、更直观。我们改进了软件的导航和组织方式，使您更容易找到所需的工具和功能。我们还添加了一系列新功能，包括主题、改进的对象操作等等。使用新 UI，您的 3D 打印体验将更快、更高效、更愉悦。

### 支持 DirectX 和 Metal

<DirectXAndMetal />

通过支持 DirectX 和 Metal，CHITUBOX Basic 现在可以渲染更逼真、更复杂的 3D 模型，具有更多细节和更逼真的照明、阴影和菲涅尔效果。这些技术还能提高性能，让您充分发挥显卡的威力。现在，在 Windows 和 macOS 平台上使用 CHITUBOX Basic 时，您可以体验到最高级别的图形性能。

### RMA

<Rma />

我们推出了新的 RMA（树脂联盟）功能。此功能使您可以轻松地为不同品牌的树脂材料选择合适的设置，它还支持云存储，因此您可以从任何地方访问您的设置。此外，此功能会不断地更新，因此您始终可以获得最新配置。有了这个新功能，您现在可以快速轻松地设置打印机，以获得最佳 3D 打印结果。

### 基本标准零件

<StandardParts />

此功能提供常用的基本形状零件库，这样您就不必花时间在互联网上搜索。
