# 要求

## 系统要求

### Windows

|          | 最低配置                                     | 推荐配置                                  |
| -------- | -------------------------------------------- | ----------------------------------------- |
| 处理器   | Intel® Core™ i5 6600K 或者 AMD Ryzen™ 5 1600 | Intel Core i7 4790 或者 AMD Ryzen 7 2700X |
| 操作系统 | Windows 7 (64 位)                            | Windows 10 (64 位)                        |
| 内存     | 16GB                                         | ≥ 32GB                                    |
| 硬盘     | 不低于 10GB 的硬盘存储空间                   | 20GB 以上的固态硬盘                       |
| 显卡     | GPU 显存：≥1GB                               | GPU 显存：≥1GB                            |
| 显示器   | 分辨率 ≥ 1280x960                            | 分辨率 ≥ 1280x1024                        |

<Admonition type="warning" title="警告">
	部分 Quadro 系列显卡可能与 CHITUBOX
	不兼容，如果您遇到此问题，可以将您的显卡型号报告至电子邮件 📧
	support@chitubox.com，这可能有助于我们分析问题。
</Admonition>

### macOS

|          | 最低配置                    | 推荐配置                         |
| -------- | --------------------------- | -------------------------------- |
| 处理器   | Intel® 或 M1 处理器         | Intel® 或 M1 处理器              |
| 操作系统 | macOS Catalina (10.15 版本) | macOS Big Sur (11.6 版本) 或以上 |
| 内存     | 16GB                        | ≥ 32GB                           |

<Admonition type="note" title="注意">
	理论上，您可以使用 Rosetta 2 在 M1 处理器的 Mac 上运行 CHITUBOX（确保您的
	Mac 版本已更新到 Big Sur）。 但是，我们不能保证以这种方式运行 CHITUBOX
	时没有性能问题。
</Admonition>

### Linux

|          | 最低配置                                       | 推荐配置                         |
| -------- | ---------------------------------------------- | -------------------------------- |
| 处理器   | 工作主频不低于 2.0GHz 的 Intel® 或 AMD® 处理器 | 工作主频不低于 3GHz 的多核处理器 |
| 操作系统 | Ubuntu 20.04                                   | Ubuntu 20.04 或更高版本          |
| 内存     | 16GB                                           | ≥ 32GB                           |
| 硬盘     | 不低于 10GB 的硬盘存储空间                     | 20GB 以上的固态硬盘              |
