# 操作

各类操作选项位于主窗口的顶部。它包括打开、保存、截图、复制、自动布局、镂空、挖洞、修复、标准件、树脂材料联盟，消息提醒和登录。

## 打开

<ChituboxOpenIcon /> “打开” 选项允许您打开 3D 网格文件、_.chitubox_
项目文件或切片文件。

## 保存

<ChituboxSaveIcon /> “保存” 选项允许您将当前项目保存为 _.chitubox_ 文件或 3D
网格文件。

## 复制

<ChituboxCopyIcon /> “复制” 选项允许用户快速制作多个 3D
模型的副本，这在打印单个模型的多个副本时非常有用。

## 镂空

<ChituboxHollowIcon /> 镂空选项有 3 种模式：“内镂空”、“外镂空”和“底部贯穿”。

**内镂空** 允许用户创建具有空心内部的 3D 模型，同时保留模型的外形。镂空选项可以帮助用户在打印 3D 模型时节省时间和材料成本，因为空心内部只需要更少的材料来打印。该功能还可用于创建具有复杂外形的轻型模型。

**外镂空** 将从模型中剥离指定厚度的壁，就像将模型进行“收缩”一样。

**底部贯穿** 与“镂空”非常相似，但会删除室底。

	<AntdImage
		src={

		}
	/>

### 镂空参数解释

-   **壁厚**：壁厚决定了墙的总厚度。
-   **精度**：精度决定了内壁上根据设定百分比生成的面数。
    <ImageContainer>
    	<ImageComparison
    		img1Src={
    			"/images/docs/en-US/chitubox-basic/2.x.x/006-operations-low-precision.png"
    		}
    		text1={"低精度"}
    		img2Src={
    			"/images/docs/en-US/chitubox-basic/2.x.x/006-operations-high-precision.png"
    		}
    		text2={"高精度"}
    	/>
    </ImageContainer>
-   **填充结构**：填充的类型。
-   **填充密度**：填充的密度。
-   **动画**：启用此选项将显示镂空过程。

## 挖洞

<ChituboxDigHoleIcon /> “挖洞”功能允许用户在几个简单的点击中创建指定形状或大小的
3D 模型中的孔。

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/zh-CN/chitubox-basic/2.x.x/006-operations-dig-hole.png"
		}
	/>
</ImageContainer>

### 挖洞参数解释

-   **形状**：孔的形状，可选择的形状有圆形、六边形和正方形。
-   **直径**：孔的直径。
-   **深度**：孔可以穿透的壁的最大厚度。

    <Admonition type="note" title="注意">
    	此功能旨在穿透模型的墙壁，因此它不会根据您设置的深度挖掘非穿透孔。
    </Admonition>

-   **垂直于**：垂直于屏幕或模型的挖掘角度。
-   **保持塞子**：是否保留打孔塞子。

## 修复

<ChituboxRepairIcon /> 修复选项将尝试修复损坏的模型。

<Admonition type="warning" title="警告">
	如果模型复杂或存在许多错误（如倒置的法向、重叠的面等），修复功能可能无法正常工作。
</Admonition>
