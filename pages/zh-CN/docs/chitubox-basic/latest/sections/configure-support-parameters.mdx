# 设置支撑参数

点击主窗口右上方的支撑图标 <ChituboxSupport />进入支撑配置界面。

你可以在这里导入 <ChituboxImportConfigIcon />，导出 <ChituboxExportConfigIcon /> 或重置 <ChituboxResetIcon /> 你的支撑配置文件。

Z 抬升高度(毫米)：模型最低点与平台之间的高度距离。

## 支撑设置

有细、中、粗三种支撑类型可选。

### 基本设置

顶部上端直径 (毫米)：顶部支撑主体上端的直径尺寸。

底阀形状：选择与打印平台接触底阀的形状。

接触点间距 (毫米)：任意两个接触点之间的最小水平距离。

角度 (°)：模型被允许添加支撑的最大角度，该角度值为模型表面与平台的夹角。

### 高级设置

#### 顶部

| 参数                | 说明                             |
| ------------------- | -------------------------------- |
| 接触形状            | 支撑顶部与模型接触的形状         |
| 接触形状直径 (毫米) | 支撑顶部和模型接触部位的直径尺寸 |
| 接触深度 (毫米)     | 支撑顶部与模型接触的深度尺寸     |
| 连接形状            | 顶部支撑主体的连接形状           |
| 上端直径 (毫米)     | 顶部支撑主体上端的直径尺寸       |
| 下端直径 (毫米)     | 顶部支撑主体下端的直径尺寸       |
| 连接长度 (毫米)     | 顶部支撑主体的长度尺寸           |

#### 中部

| 参数                        | 说明                                                                                                                       |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| 形状                        | 中部支撑主体的形状选择                                                                                                     |
| 直径 (毫米)                 | 支撑主体的直径尺寸                                                                                                         |
| 角度 (°)                    | 中部支撑与顶部支撑的最小夹角                                                                                               |
| 小柱子形状                  | 小柱支撑主体的形状                                                                                                         |
| 直径 (毫米)                 | 小柱支撑主体的直径尺寸                                                                                                     |
| 上端接触深度 (毫米)         | 小柱支撑上端与模型的接触深度尺寸                                                                                           |
| 下端接触深度 (毫米)         | 小柱支撑下端与模型的接触深度尺寸                                                                                           |
| 交叉结构的最大间距(XY)      | 相邻支撑允许自动生成交叉结构时在 XY 水平面上的最大距离值。                                                                 |
| 交叉生成起始高度 (相对平台) | 设定生成交叉结构支撑的最低起始高度值，当支撑顶部接触点相对于平台的距离值大于该设定值时，才会与相邻的支撑之间形成交叉支撑。 |

#### 底部

| 参数                | 说明                                                     |
| ------------------- | -------------------------------------------------------- |
| 平台接触形状        | 支撑与打印平台的接触形状                                 |
| 接触直径 (毫米)     | 与打印平台接触的支撑部位的直径尺寸                       |
| 厚度 (毫米)         | 与打印平台接触的支撑部位的厚度尺寸                       |
| 模型接触形状        | 当支撑底部落于模型上时，支撑下端部位与模型的接触形状     |
| 接触形状直径 (毫米) | 当支撑底部落于模型上时，支撑下端部位与模型接触的直径尺寸 |
| 接触深度 (毫米)     | 当支撑底部落于模型上时，支撑下端部位与模型的接触深度尺寸 |
| 接触点              | 当支撑底部落于模型上时，支撑下端部位与模型接触点的数量   |

#### 底阀

| 参数             | 说明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏形状         | 与打印平台接触的底阀的形状                                               |
| 底筏面积比例 (%) | 设定底阀的面积比例，该值为底阀总面积相对于模型向打印平台投影总面积的比例 |
| 底筏厚度 (毫米)  | 底阀的厚度尺寸                                                           |
| 底筏高度 (毫米)  | 底阀的高度尺寸                                                           |
| 底筏坡度 (°)     | 底阀侧面与打印平台的倾斜角度                                             |

## 自动/手动支撑

自动支撑模式：有加所有，加平台两种模式可选。
