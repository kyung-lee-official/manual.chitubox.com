## 支撑设置

你可以分别编辑支撑的<b>顶部</b>，<b>中部</b>，<b>底部</b>以及<b>底筏</b>。<br/>

### 顶部

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.top.png"
	}
/>

| 参数                 | 说明                             |
| -------------------- | -------------------------------- |
| 接触形状             | 支撑顶部与模型接触形状的选择     |
| 接触形状直径（毫米） | 支撑顶部和模型接触部位的直径尺寸 |
| 接触深度（毫米）     | 支撑顶部与模型接触的深度尺寸     |
| 连接形状             | 顶部支撑主体的连接形状           |
| 上端直径（毫米）     | 顶部支撑主体上端的直径尺寸       |
| 下端直径（毫米）     | 顶部支撑主体下端的直径尺寸       |
| 连接长度（毫米）     | 顶部支撑主体的长度尺寸           |

### 中部

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.mid.png"
	}
/>

| 参数                 | 说明                             |
| -------------------- | -------------------------------- |
| 形状                 | 中部支撑主体的形状选择           |
| 直径（毫米）         | 支撑主体的直径尺寸               |
| 角度 (°)             | 中部支撑与顶部支撑的最小夹角     |
| 小柱子形状           | 小柱支撑主体的形状               |
| 直径（毫米）         | 小柱支撑主体的直径尺寸           |
| 上端接触深度（毫米） | 小柱支撑上端与模型的接触深度尺寸 |
| 下端接触深度（毫米） | 小柱支撑下端与模型的接触深度尺寸 |

### 底部

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.bot.png"
	}
/>

| 参数             | 说明                                                     |
| ---------------- | -------------------------------------------------------- |
| 平台接触形状     | 选择支撑与打印平台的接触形状                             |
| 接触直径（毫米） | 与打印平台接触的支撑部位的直径尺寸                       |
| 厚度             | 与打印平台接触的支撑部位的厚度尺寸                       |
| 模型接触形状     | 当支撑底部落于模型上时，支撑下端部位与模型的接触形状     |
| 接触直径（毫米） | 当支撑底部落于模型上时，支撑下端部位与模型接触的直径尺寸 |
| 接触深度（毫米） | 当支撑底部落于模型上时，支撑下端部位与模型的接触深度尺寸 |
| 接触点           | 当支撑底部落于模型上时，支撑下端部位与模型接触点的数量   |

### 底阀

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-support004.raf.png"
	}
/>

| 参数             | 说明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏形状         | 选择与打印平台接触底阀的形状                                             |
| 底筏面积比例 (%) | 设定底阀的面积比例，该值为底阀总面积相对于模型向打印平台投影总面积的比例 |
| 底筏厚度（毫米） | 底阀的厚度尺寸                                                           |
| 底筏高度（毫米） | 底阀的高度尺寸                                                           |
| 底筏坡度 (°)     | 底阀侧面与打印平台的倾斜角度                                             |

## 自动/手动支撑

<p>
	<b>交叉宽度（毫米）</b>：手动添加支撑时，相邻支撑是否添加交叉结构的最大距离值，当相邻支撑距离超过该值，则不添加交叉结构。
</p>
