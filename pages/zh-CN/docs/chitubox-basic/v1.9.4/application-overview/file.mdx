	请注意，.ctb 文件是最终的切片文件，仅在大多数情况下用于打印。
	您不应该对其应用 3D
	模型操作，只有一些非模型相关的操作，如曝光时间、升降速度、像素编辑（仅限
	CHITUBOX Pro）等可用。 请始终用未切片的格式（.stl、.obj 等）修改您的模型。
</Admonition>

显示支撑辅助线： 是否在模型表面显示支撑复制线。

<Admonition type="note" title="注意">
	CHITUBOX Basic V1.8.1
	开始，为了提升用户体验，我们在加载模型时禁用了支撑辅助线计算，这使得加载过程更快。
	如果你想开启该功能，自动支撑将计算辅助线（如果删除自动添加的支撑，辅助线会显示出来，但是与
	Basic V1.8.1
	相比，只在高危区域显示）。我们推荐您使用红色高亮部分来判断风险区域。
</Admonition>

<StyledAntdImage
	src={

自动保存间隔：自动保存的时间间隔。CHITUBOX 会在**自动保存目录**（默认为日志路径）下生成一个临时的 _.chitubox_ 文件。如果 CHITUBOX 正常退出，则该临时文件会被删除。

自动保存目录：您可以手动输入需要保存的目录，也可以点击 <button className="button button--secondary">...</button> 选择文件夹作为自动保存目录。

图形质量：软件的刷新率等级，数字越大，刷新率越小。
