# 介绍

ChituManager 是一款与 CHITUBOX 切片软件配套使用的 3D 打印机管理软件，可以帮助用户轻松实现对 3D 打印机一站式管理。用户无需亲自前往打印机所在位置，即可通过软件实时查看打印机状态、远程传输文件、远程打印等操作。此外，ChituManager 还支持错误上报、历史记录统计与查看、视频监控等一系列功能，简化了 3D 打印过程，帮助用户节省时间，提高效率。
