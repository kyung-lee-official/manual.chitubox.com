# 介绍

ChituManager 是与 CHITUBOX 切片软件配套使用的一款 3D 打印机管理软件，可以帮助用户轻松实现对 3D 打印机一站式管理。无需前往打印机即可实时查看打印机状态、远程传输文件、远程打印等，简化了 3D 打印过程，帮助用户节省时间，提高效率。不仅如此，软件同时支持远程和本地局域网管理，以满足用户多样化使用场景。
