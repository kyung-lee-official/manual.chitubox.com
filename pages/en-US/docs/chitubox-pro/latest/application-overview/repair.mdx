# Repair

## Repair Panel

<AntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-repair020.png"
	}
/>

The drop-down menu on the repair panel allows you to list errors **By Model** or **By Error Type**, Error Types including **Excess Shell**, **Hole**, **Bad Side**, **Reverse**, and **Intersect**.

If **Detect** is chosen, the software will automatically detect the model and error data will also be displayed. Basic Repair is sufficient for most circumstances, whereas Advanced Repair is optimized for removing excess shells.

## Tool Bar on the left

<AntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-repair023.png"
	}
/>

When repairing a model, if you need to repair a part of the model or a specified area, we can quickly select it through the **Tool Bar on the left**. Quick selection tools including **Line**, **Brush**, **Rectangle**, **Polygon**, and **Free-form**.
