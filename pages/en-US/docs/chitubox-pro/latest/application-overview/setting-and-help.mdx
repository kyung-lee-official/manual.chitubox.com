-   Auto Save Interval <br />
    The interval of auto-saving. CHITUBOX will generate a temporary _.chitubox_ project file under the **Auto Save Path** (log path by default). If CHITUBOX exits normally, the temporary project file will be removed.

-   Auto Save Directory <br />
    You can manually enter the directory to be saved, or click <button className="button button--secondary">...</button> to select a folder as the auto-save directory.

-   File Name Format <br />
    File Name Format should starts with `%{f}_`, appended with `%{date_format}`.

-   Duplicate Format <br />
    Duplicate Name should starts with `%{f}_`, appended with `%{nnn:0:1}`. <br />
    Where: <br />
    `nnn` represents the field width, in this case, `000`. <br />
    `0` represents the start value.<br />
    `1` represents the increase step.
