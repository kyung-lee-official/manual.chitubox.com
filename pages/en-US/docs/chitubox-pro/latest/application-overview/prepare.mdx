## Add Tag

<StyledAntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare005.png"
	}
/>

Add a set text label to the specified position of the model.

Here you can tag your 3D models. This function is useful when you planning to print dozens of instances from the same 3D model but using different parameters or resins or any variate that could possibly causes differences on printing results. You can mark your 3D Models by adding tags on their surface, so you won't mixed up with printing results those are similar to each others.

Double-click the canvas to add text.

## Hollow

<StyledAntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.png"
	}
/>

The selected models will be hollowed out according to the set hollowing parameters.

<ImageContainer>
	<StyledAntdImage
		src={"/images/docs/en-US/chitubox-pro/1.x.x/gif/003hollow.gif"}
		style={{ maxWidth: 200 }}
	/>
</ImageContainer>

<ImageContainer>
	<StyledAntdImage
		src={
			"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.1.png"
		}
	/>
</ImageContainer>

-   Wall Thickness: The thickness of the chamber structure after an object was hollowed (the thickness of peeled-off part under **Outer** mode).

-   Mode:

    -   _Inner_: A closed chamber left after hollowing.

    -   _Outer_: peel off the outside of an object with the thickness of **Wall Thickness**.

    -   _Bottom through_: Similar to _Inner_, but does not keep the bottom part.

-   Infill: Enable/Disable generating infill structures.

-   Precision: Number of triangles generated on the hollowed face.

-   Hollowing process animation: Play the animation when hollowing (the real progress of hollowing subjects to the progress bar in the bottom).

-   Keep the main cavity: Only one main cavity is generated when the model is hollowed out.

    <StyledAntdImage
    	src={
    		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.2.png"
    	}

## Dig Hole

<StyledAntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare009.png"
	}
/>

The selected model is excavated according to the set hole parameters.

<StyledAntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare009.1.png"
	}
/>

-   Hole Mode: Digging a penetration or non-penetration hole.

-   Edge Number: As increasing the edges, the shape of the hole will get close to a circle.

-   Î¦: The diameter of the hole.

-   B: The maximum thickness of the wall that the hole can dig through.

-   C: The angle between the hole and the bottom of the hole.

-   Rotate: The rotation angle of the hole.

-   Keep Hole: Whether to keep the cork or not.

-   Angle: Digging angle perpendicular to the screen or to the model.

-   Forced digging hole: Whether to force digging holes or not.

-   Limited Depth B: Whether to limited depth B or not.
