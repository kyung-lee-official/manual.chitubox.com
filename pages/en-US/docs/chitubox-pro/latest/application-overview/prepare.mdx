## Hollow

<AntdImage
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.png"
	}
/>

The selected models will be hollowed out according to the set hollowing parameters.

<ImageContainer>
	<AntdImage
		src={"/images/docs/en-US/chitubox-pro/1.x.x/gif/003hollow.gif"}
		style={{ maxWidth: 200 }}
	/>
</ImageContainer>

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.1.png"
		}
	/>
</ImageContainer>

-   Wall Thickness: The thickness of the chamber structure after an object was hollowed (the thickness of peeled-off part under **Outer** mode).

-   Mode:

    -   _Inner_: A closed chamber left after hollowing.

    -   _Outer_: peel off the outside of an object with the thickness of **Wall Thickness**.

    -   _Bottom through_: Similar to _Inner_, but does not keep the bottom part.

-   Infill: Enable/Disable generating infill structures.

-   Precision: Number of triangles generated on the hollowed face.

-   Hollowing process animation: Play the animation when hollowing (the real progress of hollowing subjects to the progress bar in the bottom).

-   Keep the main cavity: Only one main cavity is generated when the model is hollowed out.

    <AntdImage
    	src={
    		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-prepare008.2.png"
