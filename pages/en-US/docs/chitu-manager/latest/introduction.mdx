# Introduction

ChituManager is a 3D printer management software that works with CHITUBOX slicer and makes it easy to manage your 3D printers in one stop. With ChituManager, you can view the status of your printers in real time, transfer files remotely, and print remotely. ChituManager also supports error reporting, historical record statistics and viewing, and video monitoring. These features simplify the 3D printing process and help users save time and improve efficiency.

## Installation

Click the ChituManager icon in the top right corner of the CHITUBOX slicer's main interface. Alternatively, the software will display a download window and prompt you to download ChituManager before proceeding when you use the Network Sending function.
