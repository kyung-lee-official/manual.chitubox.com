# Introduction

ChituManager is a 3D printer management software that comes with the CHITUBOX software. It can help users easily manage 3D printers in one stop. Users can remotely view printer status, transfer files, and print remotely without going to the printer, which simplifies the 3D printing process, helps users save time and improve efficiency. In addition, the software supports both remote and local LAN management to meet the diverse usage scenarios of users.

### RMA

We are excited to announce the launch of our new RMA (Resin Material Alliance) feature. This feature allows you to easily select the appropriate settings for different brands of resin materials.
