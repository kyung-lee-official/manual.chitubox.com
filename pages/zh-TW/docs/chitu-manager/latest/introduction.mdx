# 介紹

ChituManager 是一款與 CHITUBOX 切片軟體配套使用的 3D 列印機管理軟體，可協助使用者輕鬆實現對 3D 列印機的一站式管理。使用者無需親自前往列印機所在位置，即可透過軟體即時查看列印機狀態、傳輸檔案至列印機、以及遠端啟動列印等操作。此外，Manager 還提供了錯誤回報、歷史紀錄統計與查詢、視訊監控等多項功能，簡化了 3D 列印流程，協助使用者節省時間、提升工作效率。
