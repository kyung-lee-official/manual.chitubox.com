<Admonition type="note" title="注意">
	請注意，.ctb
	檔案是最終的切片檔案，僅在大多數情況下用於列印。您不應該對其應用 3D
	模型操作，只有一些非模型相關的操作，如曝光時間、升降速度、像素編輯（僅限
	CHITUBOX Pro）等可用。請始終用未切片的格式（.stl、.obj 等）修改您的模型。
</Admonition>

顯示支撐輔助綫：是否在模型表面顯示支撐輔助綫。

<Admonition type="note" title="注意">
	CHITUBOX Basic V1.8.1
	開始，爲了提升用戶體驗，我們在加載模型時禁用了支撐輔助綫計算，這使得加載過程更快。
	如果你想開啟該功能，自動支撐將計算輔助線（如果刪除自動添加的支撐，輔助線會顯示出來，但是與
	Basic V1.8.1
	相比，只在高危區域顯示）。我們推薦您使用紅色高亮部分來判斷風險區域。
</Admonition>

自動保存間隔：自動保存的時間間隔。CHITUBOX 會在**自動保存目錄**（默認為日誌路徑）下生成一個臨時的 _.chitubox_ 文件。如果 CHITUBOX 正常退出，則該臨時文件會被刪除。

自動保存目錄：您可以手動輸入需要保存的目錄，也可以點擊 <button className="button button--secondary">...</button> 選擇檔案夾作為自動保存目錄。
