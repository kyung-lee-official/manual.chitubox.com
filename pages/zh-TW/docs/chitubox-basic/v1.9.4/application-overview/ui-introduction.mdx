**移動**：將所選的模型移動到設定的位置。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-ui-introduction-move.png"
	}
/>

-   放在底板上：以模型的最低點為基準點，將所選模型放在底板上。

-   居中：將所選模型居中放置在列印平台上。

-   重置：將所選模型的擺放位置重置。

**旋轉**：根據指定的旋轉方向和角度旋轉所選模型。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-ui-introduction-rotate.png"
	}
/>

-   按面放平：以在模型上選定的面為基準面，將模型按基準面貼合列印平台放平。

-   組合中心：以所選的多個模型的組合的中心為旋轉中心。

**縮放**：根據指定的尺寸或縮放比例縮放所選模型。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-ui-introduction-scale.png"
	}
/>

-   鎖定比例：鎖定所選模型的 X 軸、Y 軸和 Z 軸的比例進行縮放。

-   縮放以適應：對所選模型進行縮放以適應列印平台。

-   重置：重置模型大小。

**鏡像**：对模型进行 X 轴镜像，Y 轴镜像或者 Z 轴镜像。

<AntdImage
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-ui-introduction-mirror.png"
	}
/>
