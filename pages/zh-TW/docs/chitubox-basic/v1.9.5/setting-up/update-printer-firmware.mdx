# 更新列印機韌體

## 解壓縮檔案

將更新壓縮檔內的所有檔案，解壓出來。

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-update-printer-firmware-001.png"
		}
	/>

<Admonition type="warning" title="警告">
	除了 UI 檔案和 logo 檔案，其餘檔案均不可更改名稱。
</Admonition>

## 檔案類型

| 檔案類型                                        | 説明                                                                                                                                                       | 上电能否自动读取更新 | 通过列印檔案更新 |
| ----------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------- | ---------------- |
| update.LCD, <br /> update.SH4,<br /> update.TWP | update.LCD， 是主控晶片的韌體檔案，包含韌體程式及版本號信息。<br /> 部分 ELEGOO 列印機可能會用 update.SH4, <br /> 部分 Phrozen 列印機可能會用 update.TWP。 | 是                   | 可手動升級       |
| chitu_initial_file_autorun                      | 自動運行檔案清單程式，包含需要自動更新的檔案清單。**部分列印機的升級包可能不包含此檔案**。                                                                 | 是                   | 可手動升級       |
| .gcode                                          | 機器參數類檔案，包含機器參數或螢幕校正參數等等。                                                                                                           | 否                   | 是               |
| .bin                                            | 一般情況下，指觸控螢幕 UI 檔案。                                                                                                                           | 否                   | 是               |
| .CBD                                            | FPGA 核心功能檔案。                                                                                                                                        | 否                   | 是               |
| .logo                                           | 一般情況下，指列印機開機 logo 檔案。                                                                                                                       | 否                   | 是               |

<Admonition type="tip" title="提示">
	大多數情況，您只需要更新韌體檔案和 UI 檔案。
</Admonition>

## 升級方法

將所有檔案，拷貝進隨身碟中，放置在隨身碟**根目錄**下。

<Admonition type="warning" title="警告">
	只能放在 <b>根目錄</b> 下，不要放在資料夾裏。
</Admonition>

將機器斷電，並將隨身碟插入機器的 USB 插口中，隨後機器重新上電開機。

<Admonition type="note" title="注意">
	請使用 USB 2.0, 容量 16GB (包含) 以下的隨身碟。
</Admonition>

上電後，

1. 韌體更新 <br />
   如果隨身碟內的 update.LCD 的版本，與當前主板上所搭載的版本不一致，則主板會優先並自動進行更新，此時觸控螢幕不會亮屏。 <br />
   但是能夠聽到蜂鳴器提示音，蜂鳴器提示音節奏為，嘀—嘀—嘀—嘀—嘀—嘀~~~嘀—嘀。 <br />
   （即 5 短鳴 1 長鳴 2 短鳴） <br />
   韌體更新成功後，觸控螢幕將會亮起，同時程式啟動 <br />

1. 其他檔案更新 <br />
   隨後主板會自動對剩餘的檔案進行更新（UI 檔案、logo 檔案除外），並顯示更新進度情況，當所有待更新的檔案更新完成後，程式將彈出`列印完成`的提示框。

点击 **Confirm** 按钮，随后将进入程序主页面，此时所有檔案（UI 檔案、logo 檔案除外）都已更新完毕。

<Admonition type="note" title="注意">
	如果需要更新 UI 檔案或 logo 檔案，則只要將其手動列印即可。
</Admonition>

## 更新後處理

更新完成後，先將隨身碟拔出，隨後再重新給機器上電。為了避免在列印機上使用隨身碟時不必要的反覆自動更新韌體，完成更新後請將隨身碟內的所有更新檔案從隨身碟內移除。
