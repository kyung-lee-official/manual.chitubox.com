# 操作

主視窗的頂部有各種操作選項。包括打開、儲存、截圖、複製、自動排版、鏤空、挖洞、修復、標準件、樹脂材料聯盟、消息提醒和登入。

## 打開

<ChituboxOpenIcon /> 「打開」選項允許您打開 3D 網格檔案、_.chitubox_
專案檔案或切片檔案。

## 儲存

<ChituboxSaveIcon /> 「儲存」選項允許您將當前專案儲存為 _.chitubox_ 檔案或 3D
網格檔案。

## 複製

<ChituboxCopyIcon /> 「複製」選項允許使用者快速製作多個 3D
模型的副本，這在列印單一模型的多個副本時非常有用。

## 鏤空

<ChituboxHollowIcon /> 鏤空選項有 3 種模式：「内鏤空」、「外鏤空」和「底部貫穿」。

**内鏤空** 允許用戶創建具有空心內部的 3D 模型，同時保留模型的外形。鏤空選項可以幫助用戶在列印 3D 模型時節省時間和材料成本，因為空心內部只需要更少的材料來列印。該功能還可用於創建具有複雜外形的輕型模型。

**外鏤空** 將從模型中剝離指定厚度的壁，就像將模型進行「收縮」一樣。

**底部貫穿** 與「鏤空」非常相似，但會刪除室底。

-   **壁厚**：壁厚決定了牆的總厚度。
-   **精度**：精度決定了內壁上根據設定百分比生成的面數。

    	<ImageComparison

-   **填充結構**：填充的類型。
-   **填充密度**：填充的密度。
-   **動畫**：啟用此選項將顯示鏤空過程。

## 挖洞

<ChituboxDigHoleIcon /> 「挖洞」功能允許用戶在幾個簡單的點擊中創建指定形狀或大小的3D
模型中的孔。

<ImageContainer>
	<AntdImage
		src={
			"/images/docs/zh-TW/chitubox-basic/2.x.x/006-operations-dig-hole.png"
		}
	/>
</ImageContainer>

### 挖洞參數解釋

-   **形狀**：孔的形狀，可選擇的形狀有圓形、六邊形和正方形。
-   **直徑**：孔的直徑。
-   **深度**：孔可以穿透的壁的最大厚度。

    <Admonition type="note" title="注意">
    	此功能旨在穿透模型的牆壁，因此它不會根據您設定的深度挖掘非穿透孔。
    </Admonition>

-   **垂直於**：垂直於屏幕或模型的挖掘角度。
-   **保持塞子**：是否保留打孔塞子。

## 修復

<ChituboxRepairIcon /> 修復選項會嘗試修復損壞的模型。

<Admonition type="warning" title="警告">

</Admonition>
