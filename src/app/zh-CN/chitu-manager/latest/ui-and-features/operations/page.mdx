# 操作

在打印机列表界面可以查看并点击各项功能按键对打印机进行基本的操作。

## 视频

### 打开视频

当设备在线且接入了摄像头时，<VideoSurveillance /> 视频图标出现，点击即可打开摄像头并显示视频画面。

### 延时摄影

当设备接入摄像头且支持延时摄影时，在打开设备<Link href="/zh-CN/chitu-manager/latest/ui-and-features/printer-list#详细状态">详细状态</Link>面板时可以看到“延时摄影”功能选项。若启用该选项，将对接下来的打印任务进行延时摄影。

打印任务结束后，可以打开历史打印记录列表，找到对应的打印任务，点击 <DownloadTimelapseVideos /> 图标下载延时摄影视频。

## 历史记录

点击 <HistoricalRecords/> 图标可查看当前设备的历史打印记录。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitu-manager/004-operations-historical-records.png"
	}
	style={{ maxWidth: "400px" }}
/>

历史打印任务所记录的信息包括：图片，名称，状态，进度，材料，实际消耗量，时间，总时长，以及延时摄影视频等。

## 访问设备文件

当设备在线并处于空闲状态时，点击 <DeviceFileManagement /> 文件按钮即可访问当前设备文件。

<ImageContainer
	src={"/images/docs/zh-CN/chitu-manager/004-operations-device-files.png"}
/>

如果设备文件包含可打印文件，您可以将鼠标停留在对应文件上。当出现 “立即打印” 按钮时，点击此按钮即可开始打印选中文件。

## 暂停/继续打印

当设备正处于打印状态时，打开设备详细状态面板，可以看到暂停打印按钮，点击即可暂停打印。

如果设备已处于暂停状态，可通过点击继续打印按钮恢复打印。

<ImageContainer
	src={"/images/docs/zh-CN/chitu-manager/004-operations-pause-printing.png"}
	style={{ maxWidth: "400px" }}
/>

## 停止打印

打开设备详细状态面板，点击停止打印按钮终止当前打印进程。
