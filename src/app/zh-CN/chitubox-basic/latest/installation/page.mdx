# 安装

## 系统要求

## Windows

从我们的 <SpanA href="https://www.chitubox.com/zh-Hans/download/chitubox-free">官网</SpanA> 下载安装包，双击下载的安装程序开始安装过程。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/002-installation-windows-step1.png"
	}
	style={{ maxWidth: "600px" }}
/>

点击 "下一步" 继续。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/002-installation-windows-step2.png"
	}
	style={{ maxWidth: "600px" }}

阅读最终用户许可协议，如果您同意条款，请勾选 "我接受许可协议" 选项，然后点击 "下一步" 继续。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/002-installation-windows-step3.png"
	}

	src={

	}
	style={{ maxWidth: "600px" }}

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/002-installation-windows-step5.png"
	}
	style={{ maxWidth: "600px" }}
/>

等待安装完成。

## macOS

在 macOS 上使用 CHITUBOX Basic 无需安装流程，因为它是以可转移 (portable) 的形式提供的。您只需下载 CHITUBOX Basic 的最新版本，然后将其拖放到“应用程序”文件夹中即可。
