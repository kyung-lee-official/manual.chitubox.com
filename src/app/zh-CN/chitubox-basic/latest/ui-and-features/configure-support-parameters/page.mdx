# 设置支撑参数

点击主窗口右上方的支撑图标 <ChituboxSupport />进入支撑配置界面。

## Z 抬升高度

模型最低点与打印平台之间的高度距离。默认值为 5 mm。

## 支撑设置

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/010-configure-support-parameters-support-settings.png"
	}
/>

您可以在此处 <ChituboxProfileImportIcon />**导入**、<ChituboxProfileExportIcon /> **导出**和 <ChituboxProfileResetIcon /> **重置** 支撑设置。

CHITUBOX 提供了 3 种默认的支撑设置：细、中、粗。如果它们不适合您的模型，您可以自定义支撑设置并导出以供将来使用。

## 自动支撑

自动支撑有两种模式：**+全部** 和 **+平台**。

**+全部** 允许支撑在模型表面生成，而 **+平台** 只允许支撑在平台上生成。

## 操作模式

有三种操作模式：**添加**、**删除** 和 **编辑**，他们的快捷键分别是 **A**、**D** 和 **E**。三种模式下均支持多选, 您可以按住 Ctrl 键然后点击鼠标左键并拖拽来选择操作区域。

### 添加

<ChituboxAddSupportIcon /> 点击模型添加支撑。

添加支撑有**常规模式**和**直连模式** 2 种，默认为常规模式。

直连模式允许您在模型和支撑主干之间添加支撑。它的规则如下：

-   如果您先点击模型，您可以选择将支撑另一端固定在模型上，或现有支撑的主干上。
-   如果您先点击支撑，您可以选择将支撑另一端固定在现有支撑的主干上。

如果您需要切换到直连模式，您需要先弹出支撑模式面板，

<ImageContainer
	className={"max-w-sm"}
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/010-configure-support-parameters-mode-panel.png"

/>

然后选择直连模式。

<ImageContainer

### 删除

<ChituboxDeleteSupportIcon /> 点击选择想要删除的支撑，所选支撑将变为红色，再次点击按钮或按下
Delete 键删除支撑。

### 编辑

<ChituboxEditSupportIcon /> 点击选择想要编辑的支撑，所选支撑将变为绿色，按住
Ctrl 键可以选择多个支撑，可以调整选中支撑的参数，也可以拖动支撑调整支撑位置。

编辑模式下，您可以双击支撑顶部的节点来添加节点支撑（按 Esc 键退出节点支撑）。

## 清除全部

点击此按钮，所有支撑都会被清除。
