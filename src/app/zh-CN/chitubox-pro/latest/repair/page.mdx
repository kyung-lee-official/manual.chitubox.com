# 修复

<Entry
	functionMenu={[
		<div className="flex items-center gap-1">
			修复
		</div>,
	]}
/>

修复界面包含了多种功能用于检测和修复 3D 模型中的错误，以提高模型的打印成功率。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/repair.png"
	}
/>

## 错误列表

右边的错误列表可查看检测出来的模型错误，支持检测的错误类型包括**多余壳体**，**孔洞**，**坏边**，**反向**，**相交**。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/002-repair.png"
	}
/>

点击**检测**选项，软件将会自动检测模型，并展示出模型的错误类型及对应数据。点击基本修复和高级修复可一键修复绝大多数的错误类型。

## 左侧工具栏

通过**左侧工具栏**的多种选择工具可以快速选取模型局部区域进行修复。快捷选择工具包括**笔刷**、**线**、**矩形**、**多边形**和**自由曲线**。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/003-repair.png"
	}
/>

## 翻转三角面

<Entry
	functionMenu={[
		<div className="flex items-center gap-1">
			修复 &gt; 翻转三角面
		</div>,
	]}
/>

<ImageContainer
	src={
			"/images/docs/zh-CN/chitubox-pro/2.x.x/014-repair-triangledelete.png"
	}
/>

鼠标左键点击选中需翻转的三角面，点击**翻转三角面**功能，手动修正模型表面方向错误的三角面。

<Admonition type="note" title="注意">
	当没有三角面被选中时，**删除三角形**和**翻转三角形**功能不可使用。
</Admonition>
