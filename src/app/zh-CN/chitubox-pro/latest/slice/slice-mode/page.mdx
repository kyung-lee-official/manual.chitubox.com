# 切片模式

## 单参数/多参数切片

<Entry shortcut={["Ctrl", "Shift", "S"]} />

切片是导出可打印文件前的最后一步。

点击主界面的右下方的切片模式按钮，选择 **单参数切片** 或 **多参数切片**模式，点击切片按钮进入切片预览界面。

### 单参数切片

**单参数切片** 会根据 **配置列表** 中选中的配置文件来对所有打印平台上的模型进行切片。

### 多参数切片

多参数切片是一种高级的切片功能，它允许您为不同的**模型**、不同**高度**或者不同的**横截面面积**设置不同的打印参数，从而实现更精细的控制和更高的打印质量。

<Admonition type="note" title="注意">
	目前只有 CTB 格式支持多参数切片。
</Admonition>

#### 模型

不同的模型对象使用不同的打印参数。

1. 未指定 Profile 的模型会按照**默认参数配置文件**指定的 Profile 执行。

1. 在同一打印机作业环境下修改相同名称的 Profile 文件，其余位置的同一名称的 Profile 文件会同步进行更新。

1. 在编辑差异参数时，不可编辑项（`*` 标识项）及其它未显示打印参数相关项均会按照默认的 Profile 执行。

#### 高度

不同的高度层使用不同的打印参数。

1. 未指定 Profile 的高度区间会按照指定的**默认 Profile** 执行。

1. 底层和过渡层打印栏的相关参数按照按照**默认 Profile** 执行。

1. 在编辑差异参数时，不可编辑项（`*`标识项）及其它未显示打印参数相关项均会按照默认的 Profile 执行。

#### 横截面面积

不同的横截面面积使用不同的打印参数。此处的百分比指的是横截面相对于平台的总有效面积的百分比。

1. 未指定 Profile 的截面区间会按照指定的**默认 Profile** 执行。

1. 底层和过渡层打印栏的相关参数按照按照**默认 Profile** 执行。

1. 在编辑差异参数时，不可编辑项（`*`标识项）及其它未显示打印参数相关项均会按照默认的 Profile 执行。

1. 在 **横截面面积** 模式下，使用不同的横截面来区分区间，因此，一个 Profile 是有可能被应用于多个区间的。

**底层** 和 **过渡层** 相关的设置仅在最底下的区间生效。
