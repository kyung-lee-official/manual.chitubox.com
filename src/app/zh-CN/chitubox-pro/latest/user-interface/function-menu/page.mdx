# 功能菜单

<ImageContainer src="/images/docs/zh-CN/chitubox-pro/2.x.x/function-menu.png" />

## 准备功能面板

点击功能菜单的“准备”选项切换至准备功能面板。

<ImageContainer src="/images/docs/zh-CN/chitubox-pro/2.x.x/prepare-function-panel.png" />

### 添加机器

<Link href="/zh-CN/chitubox-pro/latest/printer-management/add-and-delete-printers#添加打印机">添加一个新的机器</Link>。

### 历史记录

<Link href="/zh-CN/chitubox-pro/latest/file#打开最近的文件">访问最近打开的文件</Link>。

### 打开

打开项目文件、导入三维模型或切片文件。

### 导出

导出全部模型、合并导出或分部导出已选模型（包括支撑）。

可导出为项目文件（.chitubox/.ctp）或者模型文件。

### 保存

<Link href="/zh-CN/chitubox-pro/latest/file#保存项目文件">保存当前项目文件</Link>。

### 复制

<Link href="/zh-CN/chitubox-pro/latest/model-processing#复制">复制</Link>一个或多个所选模型。

### 拆分模型

将原组合装配模型<Link href="/zh-CN/chitubox-pro/latest/model-processing#拆分模型">拆分为独立的零件模型</Link>。

### 自动布局

将多个模型按照设定的方式进行<Link href="/zh-CN/chitubox-pro/latest/model-processing#自动布局">快速排版布局</Link>。

### 添加标签

为模型<Link href="/zh-CN/chitubox-pro/latest/model-processing#添加标签">添加设定的标签</Link>。

### 布尔操作

为选定相交的模型进行模型间的**联合**、**相交**或**相减**等<Link href="/zh-CN/chitubox-pro/latest/model-processing#布尔操作">布尔操作</Link>。

### 镂空

将所选模型<Link href="/zh-CN/chitubox-pro/latest/model-processing#镂空">镂空</Link>。

### 挖洞

在选定模型上<Link href="/zh-CN/chitubox-pro/latest/model-processing#挖洞">挖洞</Link>。

## 支撑功能面板

点击功能菜单的“支撑”选项切换至支撑功能面板。

<ImageContainer src="/images/docs/zh-CN/chitubox-pro/2.x.x/support-function-panel.png" />

### 对称支撑

为模型添加<Link href="/zh-CN/chitubox-pro/latest/support/add-edit-delete#对称支撑">对称支撑</Link>。

### 合并支撑

<Link href="/zh-CN/chitubox-pro/latest/support/add-edit-delete#合并支撑">自动合并全部或部分支撑</Link>。

### 悬空检测

<Link href="/zh-CN/chitubox-pro/latest/risk-detection#悬空检测">检测悬空区域</Link>并为其添加支撑。

## 分析/测量功能面板

点击功能菜单的“分析/测量”选项切换至分析/测量功能面板。

<ImageContainer src="/images/docs/zh-CN/chitubox-pro/2.x.x/analyze-function-panel.png" />

点击<Link href="/zh-CN/chitubox-pro/latest/analyze-and-measure">此处</Link>了解详情。

## 切片功能面板

点击主窗口右下角的“单参数切片”按钮后，对模型切片并切换至切片预览窗口。

<ImageContainer src="/images/docs/zh-CN/chitubox-pro/2.x.x/slice-function-panel.png" />

### 保存

将所有模型（带支撑）保存为项目文件或模型文件。

### 切片参数编辑

<Link href="/zh-CN/chitubox-pro/latest/slice/slice-preview#切片参数编辑">编辑和调整当前切片的相关打印参数</Link>。

### 孤岛检测

<Link href="/zh-CN/chitubox-pro/latest/risk-detection#孤岛检测">检测模型是否存在孤岛</Link>。

### 编辑像素

<Link href="/zh-CN/chitubox-pro/latest/slice/slice-preview#编辑像素">编辑切片的像素</Link>。
