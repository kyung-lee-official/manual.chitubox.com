# 支撑设置

## 底阀设置

### 模型抬升高度

模型最低点与平台之间的高度距离。

### 底阀形状

#### 滑板

| 参数             | 说明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面积比例 (%) | 设定底阀的面积比例，该值为底阀总面积相对于模型向打印平台投影总面积的比例 |
| 底筏厚度（毫米） | 底阀的厚度尺寸                                                           |
| 底筏高度（毫米） | 底阀的高度尺寸                                                           |
| 底筏坡度 (°)     | 底阀侧面与打印平台的倾斜角度                                             |

#### 十字网络

| 参数             | 说明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面积比例 (%) | 设定底阀的面积比例，该值为底阀总面积相对于模型向打印平台投影总面积的比例 |
| 底筏厚度（毫米） | 底阀的厚度尺寸                                                           |
| 网格边长（毫米） | 网格型底筏的单个网格边长                                                 |
| 网格宽度（毫米） | 网格型底筏的网格边横向宽度                                               |

#### 六角网格

| 参数             | 说明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面积比例 (%) | 设定底阀的面积比例，该值为底阀总面积相对于模型向打印平台投影总面积的比例 |
| 底筏厚度（毫米） | 底阀的厚度尺寸                                                           |
| 网格边长（毫米） | 网格型底筏的单个网格边长                                                 |
| 网格宽度（毫米） | 网格型底筏的网格边横向宽度                                               |

#### 底部相连

| 参数             | 说明           |
| ---------------- | -------------- |
| 底筏厚度（毫米） | 底阀的厚度尺寸 |

##### 增强自动支撑

添加自动支撑时，一些孤立支撑主干会自动增加辅助支撑，并生成连接结构。

-   三角强化（TR）：当选中三角强化模式，添加自动支撑时，将自动连接相邻的三个孤立支撑主干，从而形成稳定的三角结构。

-   全局强化（GR）：当选中全局强化模式，添加自动支撑时，将全面强化所有孤立支撑主干，自动形成由三角结构组合而成的更加稳定的结构，将所有支撑关联起来以更好地辅助支撑的稳定性。

## 自动支撑

### 自动支撑模式

<Entry shortcut={["Ctrl", "TAB"]} />

自动支撑有三种模式：**+全部**、**+平台** 和 **+悬空**。

**+全部** 允许支撑在模型表面生成，而 **+平台** 只允许支撑在平台上生成，**+悬空** 则可以在悬空区域添加支撑。

### 自动支撑类型

自动支撑类型有：树杈、关节、自适应、垂直、竖直和小柱。

从中选择您想要的支撑类型，点击“自动添加支撑”按钮，将自动生成对应的支撑类型。
