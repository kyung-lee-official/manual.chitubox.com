# 添加，编辑和删除

## 添加支撑

### 自动支撑

<Entry
	supportSettingsPanel={["自动支撑", "自动添加支撑"]}
	shortcut={["Shift", "S"]}
/>

根据支撑参数及模型的几何特征，自动生成支撑结构，大大节省了用户的时间。

### 手动支撑

<Entry
	supportSettingsPanel={[
		"自动支撑",
		<div className="flex items-center gap-1">
			<ChituboxAddSupportIcon />
			添加支撑
		</div>,
	]}
	shortcut={["A"]}
/>

您可以根据自己的经验和对模型的理解，手动添加支撑，以获得最佳的打印效果。

#### 支撑类型

##### 自适应支撑

<Entry manualSupportOptions={["第1个选项"]} shortcut={["1"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/adaptive-support.png"}
	style={{ width: 65 }}
/>

自适应支撑能够根据模型的具体情况，智能地生成合适的支撑。

##### 垂直支撑

<Entry manualSupportOptions={["第2个选项"]} shortcut={["2"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/vertical-support.png"}
	style={{ width: 65 }}
/>

垂直于模型表面的支撑。

##### 竖直支撑

<Entry manualSupportOptions={["第3个选项"]} shortcut={["3"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/upright-support.png"}
	style={{ width: 65 }}
/>

垂直与打印平台的支撑。

##### 树杈支撑

<Entry manualSupportOptions={["第4个选项"]} shortcut={["4"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/branch-support.png"}
	style={{ width: 65 }}
/>

树杈支撑是一种单端支撑，通常用于连接模型和另一个支撑的中间部分，或者连接两个支撑，起到稳固模型的作用。

##### 关节支撑

<Entry manualSupportOptions={["第5个选项"]} shortcut={["5"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/joint-support.png"}
	style={{ width: 65 }}
/>

关节支撑由两个支撑段组成，它与树杈支撑类似，都可用于连接模型与支撑。

##### 小柱支撑

<Entry manualSupportOptions={["第6个选项"]} shortcut={["6"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/small-pillar.png"}
	style={{ width: 65 }}
/>

这是一种细支撑，它连接模型和另一个支撑的中间部分或者连接模型表面上的两个点。适用于悬空部分较小、细节较多的模型，能够提供更精细的支撑。

<Admonition type="tip" title="提示">
	树杈、关节、小柱这三种支撑类型可开启应用支撑自动吸附功能，并可在
	“高级”支撑设置面板的**更多设置**中自定义 _吸附半径_ 和 _吸附角度_ 的参数。
</Admonition>

##### 分叉支撑

<Entry manualSupportOptions={["第7个选项"]} shortcut={["7"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/forked-support.png"}
	style={{ width: 65 }}
/>

分叉支撑的顶部与模型相连，底部分叉连接到相邻支撑的中间部分，形成稳定的支撑网络结构。

##### 树状支撑

<Entry manualSupportOptions={["第8个选项"]} shortcut={["8"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/tree-support.png"}
	style={{ width: 65 }}
/>

从主支撑分出多个分支的支撑结构，类似于树枝。树状支撑能够提供更均匀的支撑力，能够有效地防止模型变形。

##### 等高支撑

<Entry manualSupportOptions={["第9个选项"]} shortcut={["9"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/contour-support.png"}
	style={{ width: 65 }}
/>

为模型的特定高度添加支撑。

##### 法面支撑

<Entry manualSupportOptions={["第10个选项"]} shortcut={["0"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/facial-support.png"}
	style={{ width: 65 }}
/>

为模型的选定面添加支撑。

##### 边界支撑

<Entry manualSupportOptions={["第11个选项"]} shortcut={["-"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/boundary-support.png"}
	style={{ width: 65 }}
/>

为模型选定面的外边缘生成支撑，以防止悬空部分在打印过程中翘曲或断裂。

### 对称支撑

<Entry
	functionMenu={[
		"支撑",
		<div className="flex items-center gap-1">
			<SymmetricalSupport />
			添加对称支撑
		</div>,
	]}
	shortcut={["Ctrl", "Shift", "S"]}
/>

选择模型对称面，并在所选对称面两侧添加镜像对称的支撑。

### 合并支撑

<Entry
	functionMenu={[
		"支撑",
		<div className="flex items-center gap-1">
			<MergeSupport />
			合并支撑
		</div>,
	]}
	shortcut={["Ctrl", "Shift", "M"]}
/>

将全部或部分支撑进行自动合并，简化支撑结构，降低打印成本。

<Entry
	functionMenu={[
		"支撑",
		"悬空检测",
		<div className="flex items-center gap-1">
