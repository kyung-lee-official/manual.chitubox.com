# 文件

## 项目文件

| 文件类型               | 文件名后缀  | 描述                                                  |
| ---------------------- | ----------- | ----------------------------------------------------- |
| CHITUBOX 项目文件 2023 | \*.ctp      | 包含更多的模型信息 (位置，旋转等，支持更大的文件大小) |
| CHITUBOX 项目文件      | \*.chitubox | 老版本的项目文件格式，包含基本模型数据和支撑数据      |

点击主窗口右下方的 “单参数切片” 按钮，对文件进行切片。然后单击右下方的 “保存切片” 按钮即可导出切片文件。
