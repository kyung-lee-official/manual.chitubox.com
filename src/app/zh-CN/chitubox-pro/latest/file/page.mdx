### 使用插件导出切片文件

在 CHITUBOX 中，有些打印机是通过插件的形式支持的。因此，您需要在保存切片前下载并导入对应的插件文件。

| 打印机型号         | 插件                                                                                                                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Longer 3D Orange10 | <SpanA href="/plugins/Longer3d-Orange10-Windows64bit-V1.4.CHplugin" download>Windows 版本</SpanA> <br /> <SpanA href="/plugins/Longer3d-Orange10-macOS-V1.4.CHplugin" download>Mac 版本</SpanA> |
| Longer 3D Orange30 | <SpanA href="/plugins/Longer3d-Orange30-Windows64bit-V1.4.CHplugin" download>Windows 版本</SpanA> <br /> <SpanA href="/plugins/Longer3d-Orange30-macOS-V1.4.CHplugin" download>Mac 版本</SpanA> |
| Longer 3D Orange4K | <SpanA href="/plugins/Longer3D-Orange4K-Windows64bit-V1.5.CHplugin" download>Windows 版本</SpanA> <br /> <SpanA href="/plugins/Longer3d-Orange4K-macOS-V1.5.CHplugin" download>Mac 版本</SpanA> |

下载完成后，您可以通过 **左上角主菜单 > 帮助 > 插件** 打开插件窗口，然后导入插件文件。
