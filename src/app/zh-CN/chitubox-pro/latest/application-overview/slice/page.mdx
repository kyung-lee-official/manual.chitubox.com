# 切片

## 单参数和多参数

切片是导出可打印文件前的最后一步。

选择 **单参数切片** 或 **多参数切片**，点击切片按钮进入切片预览界面。

	然而，这是一个理论值，它可能会受到现实世界中许多其他因素（构建板和模型表面上的树脂残留、树脂挥发性、固化过程中的密度变化等）的影响。
</Admonition>

### 单参数切片

**单参数切片** 会根据 **切片设置列表** 中选中的默认 Profile 来对所有打印平台上的打印允许区间内的模型数据进行切片。

### 多参数切片

<Admonition type="note" title="注意">
	目前只有部分打印机支持多参数切片。
</Admonition>

**多参数切片**提供了三种模式，**模型**，**高度**和**横截面面积**。

#### 1. 模型

不同的模型对象使用不同的打印参数。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-slice-preview017.png"
	}
/>

1. 未指定 Profile 的模型会按照**默认参数配置文件**指定的 Profile 执行。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-slice-preview017.1.1.png"
	}
/>

1. 在同一打印机作业环境下修改相同名称的 Profile 文件，其余位置的同一名称的 Profile 文件会同步进行更新。

1. 在编辑差异参数时，不可编辑项（`*` 标识项）及其它未显示打印参数相关项均会按照默认的 Profile 执行。

#### 3. 横截面面积

不同的横截面面积使用不同的打印参数。此处的百分比指的是横截面相对于平台的总有效面积的百分比。

<ImageContainer

1. 未指定 Profile 的截面区间会按照指定的**默认 Profile** 执行。

1. 底层和过渡层打印栏的相关参数按照按照**默认 Profile** 执行。

1. 在编辑差异参数时，不可编辑项（`*`标识项）及其它未显示打印参数相关项均会按照默认的 Profile 执行。
