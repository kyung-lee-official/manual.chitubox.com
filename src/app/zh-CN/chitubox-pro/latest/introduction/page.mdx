# 介绍

CHITUBOX Pro 是一款功能强大的3D打印软件，可以将3D网格文件切片并转换为可打印文件。 它提供了广泛的功能和定制选项，可以极大增强您的 3D 打印体验，并大幅提高打印质量并降低打印失败的风险。 它比 Basic 版提供了更多的模型准备工具，例如分割模型、添加标签、布尔运算、添加自定义的孔洞等。
