# 介绍

CHITUBOX Pro 是一款功能强大的 3D 打印软件，可以将 3D 网格文件切片并转换为可打印文件。 它提供了广泛的功能和定制选项，可以极大增强您的 3D 打印体验，并大幅提高打印质量并降低打印失败的风险。 它比 Basic 版提供了更多的模型准备工具，例如分割模型、添加标签、布尔运算、添加自定义的孔洞等。

## 主要特性

**强大的模型处理工具**：CHITUBOX Pro 提供了更多的模型处理工具，例如分割模型、添加标签、布尔运算、添加自定义的孔洞等。

**丰富的支撑选项**：CHITUBOX Pro 支持包括自适应支撑，垂直支撑，竖直支撑，树杈支撑等多种支撑选项。

**多种风险检测**：CHITUBOX Pro 提供了多种<Link href="/zh-CN/chitubox-pro/latest/risk-detection">风险检测</Link>功能，例如空腔检测，碰撞检测，悬空检测等。

**便捷的自动化功能**：CHITUBOX Pro 提供了 <Link href="/zh-CN/chitubox-pro/latest/chitu-action">ChituAction</Link> 和 <Link href="/zh-CN/chitubox-pro/latest/support/magic-support">魔法支撑</Link>，您可以方便地编排您的模型处理流程和支撑添加流程并自动化执行。

**海量材料库**：CHITUBOX Pro 内置了树脂联盟，您可以在软件内直接下载各种树脂材料的参数。您还可以自定义材料参数并上传到云端。
