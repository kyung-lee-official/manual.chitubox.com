# 账号和密钥

## 注册账号

如果您还未注册 CHITUBOX 账号，请点击<NextLink href="https://cc.chitubox.com/register">此处</NextLink>注册。

## 密钥

一个账号可以拥有多个密钥，当您购买密钥时，密钥会自动和您的账号绑定。密钥和账号之间不可解绑。

个人版密钥支持在 2 台计算机上使用，团队版密钥支持在 5 台计算机上使用，您可以随时登录用户中心解绑已绑定的计算机。

<Admonition type="danger" title="危险">
	请勿将您的密钥告诉任何人，获取密钥后应尽快绑定账号。
</Admonition>

<Admonition type="note" title="注意">
	CHITUBOX Pro 使用在线鉴权，请确保您的计算机已经连接网络。
</Admonition>

### 激活和绑定密钥

1. 当 CHITUBOX Pro 首次启动时，登录窗口会弹出。

    <ImageContainer
    	src={
    		"/images/docs/zh-CN/chitubox-pro/2.x.x/account-and-license-management-001.png"
    	}
    />

    <Admonition type="note" title="注意">
    	登录完成后将对账号进行检测，若账号不符合试用条件（账号或设备已试用），则
    	<b>开始试用</b>
    	将不予显示。
    </Admonition>

1. 输入您的密钥：

    <ImageContainer
    	src={
    		"/images/docs/zh-CN/chitubox-pro/2.x.x/account-and-license-management-002.png"
    	}
    />

    点击**确认授权**，将激活密钥并且将其绑定到您的计算机。密钥的有效期从激活日开始计算。

    如果密钥是由打印机厂商发行的，您可能需要还需要输入您的打印机 ID 进行关联:

    <LicenseCard size={420} />

    <ImageContainer
    	src={
    		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-activate-license004.png"
    	}
    />

    如果您在 CHITUBOX 官网<NextLink href="https://cc.chitubox.com/login">登录</NextLink>您的账号，在 **我的产品** > **我的密钥** 下面应该可以看到您刚刚绑定的密钥。

    <ImageContainer
    	src={
    		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-add-your-license-to-your-account-005.png"
    	}
    />

### 将密钥和设备解绑

如果您想要将 CHITUBOX 迁移到您的新计算机，您需要先将密钥从旧计算机上解绑。

在网页端<NextLink href="https://cc.chitubox.com/login">登录您的 CHITUBOX 账号</NextLink>。

点击 **我的产品** > **我的密钥**

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-add-your-license-to-your-account-006.png"
	}
/>

点击 **已绑 PC 管理** > **解绑**

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/004-setting-up-add-your-license-to-your-account-007.png"
	}
/>

您的密钥将从旧计算机上解绑，然后您就可以重新在您的新计算机上激活您的密钥。

您也可以直接在 CHITUBOX 客户端内进行计算机解绑操作。

### 更换密钥

如果密钥证过期了，您需要在软件中手动更换新密钥。

<Entry mainMenu={["账户", "授权"]} />

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/account-and-license-management-002.png"
	}
/>

将密钥更换为您的新密钥，然后点击 **确认授权**，激活密钥并且将其绑定到您的计算机。
