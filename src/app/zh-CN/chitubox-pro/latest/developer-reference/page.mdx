## 通过命令行调用 CHITUBOX Pro 并加载指定的网格文件

### Windows

`executable file [file ...]`

例子：

```bash
'C:\Program Files\CHITUBOX Pro\CHITUBOX Pro.exe' 'C:\Users\user\Documents\3D Models\test1.stl' 'C:\Users\user\Documents\3D Models\test2.stl'
```
