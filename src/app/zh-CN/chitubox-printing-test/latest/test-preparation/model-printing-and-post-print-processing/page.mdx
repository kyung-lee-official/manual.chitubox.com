# 模型打印及打印后处理

## 模型与打印平台分离

模型与打印平台分离时需要注意避免将打印模型损坏，特别需要注意需要用分离铲刀分离脆性材料时的打印模型的脆性断裂以及柔性材料分离时模型的强度不足导致的损坏。

## 模型清洗和烘干

清洗模型时需要确保表面残留树脂液体完全被清洗，被清洗的模型不得因为清洗浸泡时间过长导致固体模型被溶剂溶解或者发生二次反应。
  
清洗后的模型需要进行充分的烘干处理后才可以进一步进入下一环节的后固化处理。

## 模型后固化

合适的模型后处理时间有助于打印模型获取更好的反应充分度，进一步提升模型的反应充分度；过长时间的后固化处理可能会导致模型发生较大的变形。

## 模型存储

由于本指导说明主要针对光固化成型的模型数据，所以打印模型的基本保存需要满足基本的遮光保存条件。
