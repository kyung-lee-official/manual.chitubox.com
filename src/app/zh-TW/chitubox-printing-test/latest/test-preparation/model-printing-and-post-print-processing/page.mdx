# 模型列印及列印後處理

## 模型與列印平台分離

模型與列印平台分離時需注意避免將列印模型損壞，特別是使用分離鏟刀分離脆性材料時的列印模型的脆性斷裂，以及柔性材料分離時模型強度不足導致的損壞。

## 模型清洗和烘乾

清洗模型時，需確保表面殘留樹脂液體完全被清洗，被清洗的模型不得因清洗浸泡時間過長，導致固體模型被溶劑溶解或發生二次反應。

清洗後的模型需進行充分的烘乾處理後，才可以進一步進入下一環節的後固化處理。

## 模型後固化

適當的模型後處理時間有助於列印模型獲取更好的反應充分度，進一步提升模型的反應充分度；過長時間的后固化處理可能會導致模型發生較大的變形。

## 模型儲存

由於本指導說明主要針對光固化成型的模型數據，所以列印模型的基本保存需滿足基本的遮光保存條件。
