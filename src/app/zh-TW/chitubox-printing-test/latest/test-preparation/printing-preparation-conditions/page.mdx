# 列印準備條件

## 列印環境

1. 列印驗證室內環境溫度需穩定控制在 18~28℃。
2. 列印驗證室內環境濕度需控制不超過 60%。
3. 列印驗證過程列印機需要遠離熱源、水源及震動源。
4. 列印驗證過程室內需要保證定期的空氣流通，減少和避免列印室內人員吸入樹脂液體散發的氣味。
5. 列印驗證過程室內需保持無光或者弱藍光狀態，弱光光源推薦使用黃光燈，全程列印過程需要確保列印機的遮光罩完全閉合。
6. 列印驗證過程供電電源需滿足接地要求，且不能和大電流設備共享接線電路，列印過程需要確保供電電壓的穩定性。
7. 列印驗證過程請勿將水和酒精等揮發性溶劑放置於列印室內，避免列印成型過程產生干擾性品質問題。
8. 列印驗證過程供電電壓和電流需要嚴格按照列印機設備的供電要求進行執行。

## 列印機

1. 列印機需要安裝放置到堅硬水平的區域。
2. 檢測並確認離型膜無破損、離型膜乾淨、離型膜張緊正常。
3. 檢測並確認光源和圖像顯示裝置正常。
4. 檢測並確認正常列印使用過程光源和成像區域完全點亮後對應的光強數值。
5. 檢測並確認與離型膜貼合成像工作區乾淨無雜質。
6. 檢測並確認列印機列印平台鎖緊正常。
7. 檢測並確認列印平台無殘留物體。
8. 檢測並確認列印平台調平正常。
9. 檢測並確認樹脂槽正確安裝鎖緊。

進行列印機的確認時需要通過上述注意事項的第 4 條，進行列印機對應光強的抽樣測試確認。

對應光強抽樣測試驗證所選取的光強測量裝置需要使用下述設備並嚴格按照對應的使用指導說明，完成對應光強數值的測試。

<Admonition type="warning" title="警告">
	光強測量裝置需要統一按照對應列印機的使用波段進行測量。
</Admonition>

對於選取的列印機產品，需要按照如下圖示，固定間隔距離進行對應光強數值的測量記錄，對應區域未完全指定對應的測量位置，需要滿足光強測量裝置與圖像成型面的完全貼合，對應區域的光強數值需要分別測量並記錄5次，並根據對應點位測量數值求取對應的平均數值。

## 列印材料

1. 檢測並確認列印材料正常無過期。
2. 檢測並確認樹脂槽內加入的列印材料均勻混合無雜質。
3. 檢測並確認樹脂槽內加入的列印材料充足並不會在列印過程溢出。

#### 單模型排版

<Admonition type={"note"} title="注意">
	單模型重複多次調整參數排版測試驗證，需要確保列印模型相對同一列印機的相同位置，示例位置如下。
</Admonition>

<ImageContainer src="/images/docs/zh-CN/chitubox-printing-test/1.x.x/test-preparation/single-model_001.png" />

<Admonition type={"tip"} title="提示">
	對應模型相對平台的位置可以選中對應的模型，可觸發移動按鍵，並點擊對應的模型進行查看確認。
</Admonition>

#### 多模型排版

<Admonition type={"note"} title="注意">
	多模型重複多次進行列印測試驗證時，需要確保列印模型相對同一列印機的位置保持相同。
</Admonition>

#### 支撐參數的調整

部分测试验证模型具有 CHITUBOX 对应的粗/中/细三种类型的支撑模型，可有效测试对应的粗、中、细支撑时对应的接触点间距的参数有效性，其它参数的支撑结构的测试也可通过模型支撑结构的二次编辑和设置进行测试验证。
