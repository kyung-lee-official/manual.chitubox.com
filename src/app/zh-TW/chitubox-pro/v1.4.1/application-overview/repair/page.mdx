# 修復

## 修復面板

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-repair020.png"
	}
/>

修復面板上的下拉菜單可以選擇 **按模型顯示** 或 **按錯誤類型顯示**，檢測的錯誤類型包括 **多餘殼體**、**孔洞**、**壞邊**、**反向**、**相交**。

如果選中**檢測**選項，軟體將會自動檢測模型，並展示出模型的錯誤數據。基本修復可以完成絕大多數的修復任務，高級修復是針對清除多餘殼體優化的修復。

## 左側工具欄

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-repair023.png"
	}
/>

在修復模型時，如需對模型局部或指定區域進行修復，我們可以通過**左側工具欄**來快速選取。快捷選擇工具包括**線**、**筆刷**、**矩形**、**多邊形**和**自由曲線**。
