# 修複

<Entry
	functionMenu={[
		<div className="flex items-center gap-1">
			修複
		</div>,
	]}
/>

修復介麵包含了多種功能用於檢測和修復 3D 模型中的錯誤，以提高模型的列印成功率。

<ImageContainer
	src={
		"/images/docs/zh-TW/chitubox-pro/2.x.x/repair.png"
	}
/>

## 錯誤列表

右邊的錯誤列表可查看檢測出來的模型錯誤，支持檢測的錯誤類型包括**多餘殼體**，**孔洞**，**壞邊**，**反向**，**相交**。

<ImageContainer
	src={
		"/images/docs/zh-TW/chitubox-basic/2.x.x/002-repair.png"
	}
/>

點擊**檢測**選項，軟件將會自動檢測模型，並展示出模型的錯誤類型及對應數據。點擊基本修複和高級修複可一鍵修複絶大多數的錯誤類型。

## 左側工具欄

通過**左側工具欄**的多種選擇工具可以快速選取模型局部區域進行修複。快捷選擇工具包括**筆刷**、**線**、**矩形**、**多邊形**和**自由曲線**。
<ImageContainer
	src={
		"/images/docs/zh-TW/chitubox-basic/2.x.x/003-repair.png"
	}
/>
