# 切片面板

切片面板位於主界面的右下角。包含切片設定、切片模式和單參數/多參數切片按鈕。

## 切片設定

開啟目前列印機的<Link href="/zh-TW/chitubox-pro/latest/printer-management/printer-configuration">切片設定</Link>。

## 切片模式

选择单参数或多参数<Link href="/zh-TW/chitubox-pro/latest/slice/slice-mode">切片模式</Link>。 若所選列印機不支援多參數切片，則不支援修改。

## 單參數/多參數切片

點擊此按鈕對項目切片，進入<Link href="/zh-TW/chitubox-pro/latest/slice/slice-preview">切片預覽</Link>界面。
