### 支撐組

按照模型及支撐類型分組支撐，您可以快速預覽所有支撐信息、批量編輯和刪除支撐。

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/toolbar-support-group.png"}
/>

### 選擇工具

可以使用不同的工具來選擇需要添加支撐的模型區域，也可以選擇已生成的支撐進行編輯或刪除。有五種選擇工具可選：筆刷、線、矩形、多邊形和自由曲線。
