# 支撐設定

## 底閥設定

### 模型抬升高度

模型最低點與列印平台之間的高度距離。

### 底閥形狀

#### 滑板

| 參數             | 説明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面積比例 (%) | 設定底筏的面積比例，該值為底筏總面積相對於模型向打印平台投影總面積的比例 |
| 底筏厚度（毫米） | 底筏的厚度尺寸                                                           |
| 底筏高度（毫米） | 底筏的高度尺寸                                                           |
| 底筏坡度 (°)     | 底筏側面與打印平台的傾斜角度                                             |

#### 十字网络

| 參數             | 説明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面積比例 (%) | 設定底筏的面積比例，該值為底筏總面積相對於模型向打印平台投影總面積的比例 |
| 底筏厚度（毫米） | 底筏的厚度尺寸                                                           |
| 網格邊長（毫米） | 網格型底筏的單個網格邊長                                                 |
| 網格寬度（毫米） | 網格型底筏的網格邊橫向寬度                                               |

#### 六角网格

| 參數             | 説明                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| 底筏面積比例 (%) | 設定底筏的面積比例，該值為底筏總面積相對於模型向打印平台投影總面積的比例 |
| 底筏厚度（毫米） | 底筏的厚度尺寸                                                           |
| 網格邊長（毫米） | 網格型底筏的單個網格邊長                                                 |
| 網格寬度（毫米） | 網格型底筏的網格邊橫向寬度                                               |

#### 底部相连

| 參數             | 説明           |
| ---------------- | -------------- |
| 底筏厚度（毫米） | 底筏的厚度尺寸 |

##### 增强自動支撐

添加自動支撐時，一些孤立支撐主幹會自動增加輔助支撐，並生成連接結構。

-   三角强化（TR）：當選中三角強化模式，添加自動支撐時，將自動連接相鄰的三個孤立支撐主幹，從而形成穩定的三角結構。

-   全局强化（GR）：當選中全局強化模式，添加自動支撐時，將全面強化所有孤立支撐主幹，自動形成由三角結構組合而成的更加穩定的結構，將所有支撐關聯起來以更好地輔助支撐的穩定性。

## 自動支撐

### 自動支撐模式

<Entry shortcut={["Ctrl", "TAB"]} />

自動支撐有三种模式：**+全部**、**+平台** 和 **+懸空**。

**+全部** 允許支撐在模型表面生成，而 **+平台** 只允許支撐在平台上生成，**+懸空** 則可以在懸空区域添加支撐。

### 自動支撐类型

自動支撐類型有：樹杈、關節、自適應、垂直、豎直和小柱。

從中選擇您想要的支撐類型，點擊「自動添加支撐」按鈕，將自動生成對應的支撐類型。
