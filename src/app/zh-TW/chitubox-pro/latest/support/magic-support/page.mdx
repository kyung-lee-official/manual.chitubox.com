## 流程管理

### 新增/刪除步驟

-   新增步驟：點擊步驟卡片底部的加號 **+** ，新增新的自動支撐步驟。

-   刪除步驟：點擊單個步驟卡片右側的減號 **-**， 刪除不需要的步驟。

### 步驟設定

| 參數         | 説明                   |
| ------------ | ---------------------- |
| 支撐參數     | 選擇支撐參數配置文件。 |
| 自動支撐模式 | 選擇自動支撐模式。     |
| 自動支撐類型 | 選擇自動支撐類型。     |

### 合併支撐

開啟合併支撐選項後，將自動合併已添加的支撐。
