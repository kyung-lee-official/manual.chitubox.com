# 添加、編輯和刪除

## 添加支撐

### 自動支撐

<Entry
	supportSettingsPanel={["自動支撐", "自動添加支撐"]}
	shortcut={["Shift", "S"]}
/>

根據支撐參數及模型的幾何特徵，自動生成支撐結構，大大節省了用戶的時間。

### 手動支撐

<Entry
	supportSettingsPanel={[
		"自動支撐",
		<div className="flex items-center gap-1">
			<ChituboxAddSupportIcon />
			添加支撐
		</div>,
	]}
	shortcut={["A"]}
/>

您可以根據自己的經驗和對模型的理解，手動添加支撐，以獲得最佳的列印效果。

#### 支撐類型

##### 自適應支撐

<Entry manualSupportOptions={["第1個選項"]} shortcut={["1"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/adaptive-support.png"}
	style={{ width: 65 }}
/>

自適應支撐能夠根據模型的具體情況，智能地生成合適的支撐。

##### 垂直支撐

<Entry manualSupportOptions={["第2個選項"]} shortcut={["2"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/vertical-support.png"}
	style={{ width: 65 }}
/>

垂直於模型表面的支撐。

##### 豎直支撐

<Entry manualSupportOptions={["第3個選項"]} shortcut={["3"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/upright-support.png"}
	style={{ width: 65 }}
/>

垂直與列印平台的支撐。

##### 樹杈支撐

<Entry manualSupportOptions={["第4個選項"]} shortcut={["4"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/branch-support.png"}
	style={{ width: 65 }}
/>

樹杈支撐是一種單端支撐，通常用於連接模型和另一個支撐的中間部分，或是連接兩個支撐，起到穩固模型的作用。

##### 關節支撐

<Entry manualSupportOptions={["第5個選項"]} shortcut={["5"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/joint-support.png"}
	style={{ width: 65 }}
/>

關節支撐由兩個支撐段組成，它與樹杈支撐類似，都可用於連接模型與支撐。

##### 小柱支撐

<Entry manualSupportOptions={["第6個選項"]} shortcut={["6"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/small-pillar.png"}
	style={{ width: 65 }}
/>

這是一種細支撐，它連接模型和另一個支撐的中間部分或是連接模型表面上的兩個點。適用於懸空部分較小、細節較多的模型，能夠提供更精細的支撐。

<Admonition type="tip" title="提示">
	樹杈、關節、小柱這三種支撐類型可開啟應用支撐自動吸附功能，並可在
	「高級」支撐設置面板的**更多設置**中自訂 _吸附半徑_ 和 _吸附角度_ 的參數。
</Admonition>

##### 分叉支撐

<Entry manualSupportOptions={["第7個選項"]} shortcut={["7"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/forked-support.png"}
	style={{ width: 65 }}
/>

分叉支撐的頂部與模型相連，底部分叉連接到相鄰支撐的中間部分，形成穩定的支撐網絡結構。

##### 樹狀支撐

<Entry manualSupportOptions={["第8個選項"]} shortcut={["8"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/tree-support.png"}
	style={{ width: 65 }}
/>

從主支撐分出多个分支的支撐結構，類似於樹枝。樹狀支撐能夠提供更均勻的支撐力，能夠有效地防止模型變形。

##### 等高支撐

<Entry manualSupportOptions={["第9個選項"]} shortcut={["9"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/contour-support.png"}
	style={{ width: 65 }}
/>

為模型的特定高度添加支撐。

##### 法面支撐

<Entry manualSupportOptions={["第10個選項"]} shortcut={["0"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/facial-support.png"}
	style={{ width: 65 }}
/>

為模型的選定面添加支撐。

##### 邊界支撐

<Entry manualSupportOptions={["第11個選項"]} shortcut={["-"]} />

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/boundary-support.png"}
	style={{ width: 65 }}
/>

為模型選定面的外邊緣生成支撐，以防止懸空部分在列印過程中翹曲或斷裂。

### 對稱支撐

<Entry
	functionMenu={[
		"支撐",
		<div className="flex items-center gap-1">
			<SymmetricalSupport />
			添加对称支撐
		</div>,
	]}
	shortcut={["Ctrl", "Shift", "S"]}
/>

選擇模型對稱面，並在所選對稱面兩側添加鏡像對稱的支撐。

### 合併支撐

<Entry
	functionMenu={[
		"支撐",
		<div className="flex items-center gap-1">
			<MergeSupport />
			合併支撐
		</div>,
	]}
	shortcut={["Ctrl", "Shift", "M"]}
/>

將全部或部分支撐進行自動合併，簡化支撐結構，降低列印成本。

### 懸空支撐

<Entry
	functionMenu={[
		"支撐",
		"懸空檢測",
		<div className="flex items-center gap-1">
			<AddSupportToSuspendedArea />
			添加懸空支撐
		</div>,
	]}
/>

<Link href="/zh-TW/chitubox-pro/latest/risk-detection#懸空檢測">懸空檢測</Link>後，為懸空區域手動或自動添加支撐結構，以防止懸空部分在列印過程中變形或斷裂。

## 編輯支撐

<Entry
	supportSettingsPanel={[
		"自動支撐",
		<div className="flex items-center gap-1">
			<ChituboxEditSupportIcon />
			編輯支撐
		</div>,
	]}
	shortcut={["E"]}
/>

對已添加的支撐進行調整，包括修改支撐的大小、密度和位置，以優化列印效果。
