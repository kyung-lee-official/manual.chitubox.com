# 風險檢測

## 空腔檢測

<CavityDetection /> 空腔檢測位於機器標籤右側。

空腔檢測可以幫助用戶識別 3D 列印模型中的任何封閉空腔，例如封閉的口袋或空洞，並且用戶可以向模型添加孔以避免真空效應。

## 懸空檢測

懸空檢測在支撐視圖中可用，點擊功能選單的「支撐」選項切換到支撐視圖。懸空檢測面板在支撐功能面板上。

<Entry
	functionMenu={["支撐", "懸空檢測面板"]}
	shortcut={["Ctrl", "Shift", "D"]}
/>

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/risk-detection-suspend-detection.png"
	}
/>

它提供了三個準確度等級：低、中和高。 精度等級越高，檢測結果越準確，但檢測時間越長。 點擊「刷新」按鈕 <ChituboxRefreshIcon />，可以刷新檢測結果。

點擊 「懸空區域」 面板中的箭頭將自動放大模型中的相應區域。點擊<AddSupportToSuspendedArea /> 按鈕可以為懸空區域自動添加支撐。

## 孤島檢測

<Entry
	functionMenu={["切片功能面板", "孤島檢測"]}
	islandsDetectionPanel={["孤島信息", "開始檢測"]}
	shortcut={["Ctrl", "Shift", "D"]}
/>

孤島檢測與懸空檢測類似，但是它是通過迭代 2D 圖層而不是 3D 模型的方式來檢測孤島。要使用孤島檢測，需要先點擊主面板右側的 **單參數切片** 按鈕對模型進行切片。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/single-parameter-slice-button.png"
	}
/>

切片模型後，您會進入預覽窗口，並在預覽窗口頂部看到 **孤島檢測** 按鈕。

### 查看孤島

點擊預覽窗口頂部的 <ChituboxIslandsDetectionIcon /> 檢測孤島按鈕或右側的「開始檢測」按鈕將自動檢測模型中的孤島。

如果檢測出孤島，在孤島檢測面板中將顯示孤島信息列表，孤島信息包括孤島所在層數、座標。

點擊單個孤島座標將在預覽窗口中以紅色突出顯示已檢測到的孤島。

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/2.x.x/risk-detection-islands.png"}
/>

<Admonition type="tip" title="提示">
	滑鼠左鍵點擊對應孤島座標，可切換預覽圖為孤島**局部模式**或**全局模式**。
</Admonition>

### 刪除孤島

點擊右側孤島檢測面板上的**刪除所有孤島**按鈕會將當前檢測的孤島全部刪除。通常，您不希望直接使用此功能，而是希望返回並為已檢測到的孤島添加支撐。

點擊刪除按鈕 <DeleteIslandsIcon /> 刪除所選層數的所有孤島或刪除單個已檢測到的孤島。

刪除孤島後，點擊「刷新」按鈕 <ChituboxRefreshIcon />，可以刷新檢測結果。
