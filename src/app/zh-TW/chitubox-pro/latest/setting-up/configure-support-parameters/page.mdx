#### 增强自動支撐

當添加自動支撐時，一些孤立的支撐主幹會自動增加輔助支撐，並生成連接結構。

-   三角強化（TR）：當選中三角強化模式，添加自動支撐時，將自動連接相鄰的三個孤立的支撐主幹，從而形成穩定的三角結構。

-   全局強化（GR）：當選中全局強化模式，添加自動支撐時，將全面強化所有孤立的支撐主幹，自動形成由三角結構組合而成的更加穩定的結構，將所有支撐關聯起來以更好地輔助支撐的穩定性。
