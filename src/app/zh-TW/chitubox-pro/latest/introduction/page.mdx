# 介紹

CHITUBOX Pro 是一款功能強大的 3D 列印軟體，可以將 3D 網格檔案切片並轉換為可列印檔案。 它提供了廣泛的功能和定制選項，可以極大增強您的 3D 列印體驗，並大幅提高列印品質並降低列印失敗的風險。 它比 Basic 版提供了更多的模型準備工具，例如分割模型、添加標籤、布爾運算、添加自訂的孔洞等。

## 主要特性

**強大的模型處理工具**：CHITUBOX Pro 提供了更多的模型處理工具，例如分割模型、添加標籤、布爾運算、添加自訂的孔洞等。

**豐富的支撐選項**：CHITUBOX Pro 支持包括自適應支撐，垂直支撐，豎直支撐，樹杈支撐等多種支撐選項。

**多种风险检测**：CHITUBOX Pro 提供了多種<NextLink href="/zh-CN/chitubox-pro/latest/risk-detection">風險檢測</NextLink>功能，例如空腔檢測，碰撞檢測，懸空檢測等。

**便捷的自動化功能**：CHITUBOX Pro 提供了 <NextLink href="/zh-CN/chitubox-pro/latest/chitu-action">ChituAction</NextLink> 和 <NextLink href="/zh-CN/chitubox-pro/latest/support/magic-support">魔法支撐</NextLink>，您可以方便地編排您的模型處理流程和支撐添加流程並自動化執行。

**海量材料庫**：CHITUBOX Pro 內建了樹脂聯盟，您可以在軟體內直接下載各種樹脂材料的參數。您還可以自訂材料參數並上傳到雲端。
