# 切片模式

## 單參數/多參數切片

<Entry shortcut={["Ctrl", "Shift", "S"]} />

切片是导出可列印文件前的最後一步。

點擊主介面的右下方的切片模式按鈕，選擇**單參數切片**或**多參數切片**模式，點擊切片按鈕進入切片預覽介面。

### 單參數切片

**單參數切片** 會根據 **配置列表** 中選中的配置文件來對所有列印平台上的模型進行切片。

### 多參數切片

多參數切片是一種進階的切片功能，它允許您為不同的**模型**、不同**高度**或者不同的**橫截面面積**設定不同的列印參數，從而實現更精細的控制和更高的列印品質。

<Admonition type="note" title="注意">
	目前只有 .ctb 格式支援多參數切片。
</Admonition>

#### 模型

不同的模型物件使用不同的列印參數。

1. 未指定 Profile 的模型會按照**預設參數配置文件**指定的 Profile 執行。

1. 在同一列印機作業環境下修改相同名稱的 Profile 檔案，其餘位置的同一名稱的 Profile 檔案會同步進行更新。

1. 在編輯差異參數時，不可編輯項（* 标识項）及其他未顯示列印參數相關項均會按照預設的 Profile 執行。

#### 高度

不同的高度層使用不同的列印參數。

1. 未指定 Profile 的高度區間會按照指定的**預設 Profile** 執行。

1. 底層和過渡層列印欄的相關參數按照按照**預設 Profile** 執行。

1. 在編輯差異參數時，不可編輯項（`*`标识項）及其他未顯示列印參數相關項均會按照預設的 Profile 執行。

#### 橫截面面積

不同的橫截面面積使用不同的列印參數。此處的百分比指的是橫截面相對於平台的總有效面積的百分比。

1. 未指定 Profile 的截面區間會按照指定的**預設 Profile** 執行。

1. 底層和過渡層列印欄的相關參數按照按照**預設 Profile** 執行。

1. 在編輯差異參數時，不可編輯項（`*`标识項）及其他未顯示列印參數相關項均會按照預設的 Profile 執行。

1. 在 **橫截面面積** 模式下，使用不同的橫截面來區分區間，因此，一個 Profile 是有可能被應用於多個區間的。

**底層** 和 **過渡層** 相關的設定僅在最底下的區間生效。
