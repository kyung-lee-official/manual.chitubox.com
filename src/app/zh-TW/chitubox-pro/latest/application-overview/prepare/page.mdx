## 鏤空

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare008.png"
	}
/>

將選定的模型按照設定的鏤空參數鏤空。

<ImageContainer
	src={"/images/docs/zh-CN/chitubox-pro/1.x.x/gif/003hollow.gif"}
	style={{ maxWidth: 200 }}
/>

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare008.1.png"
	}
/>

-   壁厚：鏤空時被操作物件剩餘的腔體厚度（**外鏤空** 模式下時為被削去的厚度）。
-   模式：
    -   _内鏤空_：鏤空後剩餘一個封閉腔體。
    -   _外鏤空_：削去對物件外層，厚度為**壁厚**。
    -   _底部貫穿_：和*内鏤空*類似，但不保留底部。
-   填充：開啟/關閉生成内部填充結構。
-   精度：被鏤空面生成的三角面數量。
-   動畫：鏤空時展示鏤空動畫（實際鏤空進度以底部進度條為準）。
-   僅保留主腔：在模型鏤空時只生成一個主腔體。

## 挖洞

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare009.png"
	}
/>

將選定的模型按照設定的挖洞參數挖洞。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-prepare021.png"
	}
/>

-   挖洞模式：有貫穿和非貫穿模式兩種。
-   多邊形邊數：邊數越大，孔的形狀將越接近圓形。
-   Φ：孔的直徑。
-   B：孔能夠穿透的最大深度。
-   C：孔洞柱體與孔底部形成的夾角。
-   旋轉：孔的旋轉角度。
-   保留孔：開啟/關閉保留打孔塞子。
-   角度：以垂直於模型/相機透視角度進行挖洞。
-   強制挖洞：開啟/關閉強制挖洞。
-   限制深度 B：開啟/關閉限制挖洞深度 B。
