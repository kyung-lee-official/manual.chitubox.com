## 打開

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-start003.png"
	}
/>

開啟專案檔案、三維模型或切片檔案。

<Admonition type="note" title="注意">
	請注意，_.ctb_
	檔案是最終的切片檔案，僅在大多數情況下用於列印。您不應該對其應用 3D
	模型操作，只有一些非模型相關的操作，如曝光時間、升降速度、像素編輯（僅限
	CHITUBOX Pro）等可用。請始終用未切片的格式（_.stl_、_.obj_
	等）修改您的模型。
</Admonition>

## 模型替換

選取模型後，點擊滑鼠右鍵，再選擇“替換模型”。您可以對已經新增支撐的模型進行替換，並保留已有的支撐。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-start001.gif"
	}
/>

<Admonition type="note" title="注意">
	新模型主要是做局部細節的調整，整體性上與舊模型保持相似。
</Admonition>
