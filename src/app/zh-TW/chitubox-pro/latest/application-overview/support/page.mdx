## 左側工具欄

在**支撐**介面下，左側工具欄也相應地有所變化。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support020.png"
	}
/>

使用這些工具選擇特定區域，即可將添加，刪除和編輯操作應用於其中的支撐。

以**矩形**爲例。

-   添加

    選中區域後，點擊 “+ 全部” 或 “+ 平台” 即可以對該區域自動添加支撐。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support020.1.gif"
	}
/>

-   刪除

    刪除模式下被選中的支撐呈紅色，表示可以被刪除。

<ImageContainer

		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support020.4.gif"
	}
/>

-   編輯

    編輯模式下被選取的支撐呈綠色，表示可以被編輯。

## 支撐類型

在工作視窗底部，支撐種類面板提供了不同種類的支撐以滿足你的需求。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support015.png"
	}
/>

### 智能

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-01.png"
	}
/>

這是最常用的支撐，根據您的支撐設定自動調整。

### 法向

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-02.png"
	}
/>

這種支撐的頂部平行於模型表面的法線，垂直於模型的表面。

### 竖直

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-03.png"
	}
/>

這種類型的支撐的頂部垂直於列印平台。

### 直連

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-04.png"
	}
/>

這是單段支撐，它連接模型和另一個支撐的中段。允許兩端都添加在支撐上。

### 兩點相連

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-05.png"
	}
/>

這是二段支架，它連接模型和另一個支撐的中閒部分。

### 小柱子

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-06.png"
	}
/>

這是一種細支撐，它連接模型和另一個支撐的中段或者連接模型表面上的兩個點。

<Admonition type="tip" title="提示">
	直連、兩點相連、小柱子這三種支撐類型可開啟應用支撐自動吸附功能，並可在
	“進階” 支撐設定面板中自定義 _吸附半徑_ 和 _吸附角度_ 的參數。
</Admonition>

### 分叉

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-07.png"
	}
/>

這種支撐的頂部與模型相連，底部分為兩段並連接到相鄰的兩個支撐的中間部分。

### 等高分佈

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-08.png"
	}
/>

這將生成一堆支撐，其頂部以相同的輪廓連接到模型。

### 樹狀支撐

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-support-type-09.png"
	}
/>

允許您添加樹狀支撐。
