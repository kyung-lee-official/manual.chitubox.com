# 切片

## 單參數和多參數

切片是將模型文件轉換為可列印文件的最後一步。

選擇 **單參數切片** 或 **多參數切片**，然後點擊切片按鈕進入切片預覽界面。

	然而，這只是一個理論值，它可能會受到現實世界中許多其他因素（構建板和模型表面上的樹脂殘留、樹脂揮發性、固化過程中的密度變化等）的影響。
</Admonition>

### 單參數切片

**單參數切片** 會根據 **切片設定列表** 中選中的默認 Profile 來對所有列印平台上的列印允許區間內的模型數據進行切片。

### 多參數切片

<Admonition type="note" title="注意">
	目前只有部分列印機支持多參數切片。
</Admonition>

**多參數切片**提供了三種模式，**模型**、**高度**和**橫截面面積**。

#### 1. 模型

不同的模型對象使用不同的列印參數。

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-slice-preview017.png"
	}
/>

1. 未指定 Profile 的模型會按照**默認參數配置文件**指定的 Profile 執行。

    <ImageContainer
    	src={
    		"/images/docs/zh-CN/chitubox-pro/1.x.x/006-application-overview-slice-preview017.1.1.png"
    	}
    />

1. 在同一列印機作業環境下修改相同名稱的 Profile 文件，其餘位置的相同名稱的 Profile 文件會同步進行更新。

1. 在編輯差異參數時，不可編輯項（`*` 標識項）及其它未顯示列印參數相關項均會按照默認的 Profile 執行。

    <ImageContainer

#### 3. 橫截面面積

不同的橫截面面積使用不同的列印參數。此處的百分比指的是橫截面相對於平台的總有效面積的百分比。

	src={

	}
/>

1. 未指定 Profile 的截面區間會按照指定的**默認 Profile** 執行。

1. 底層和過渡層列印欄的相關參數按照按照**默認 Profile** 執行。

1. 在編輯差異參數時，不可編輯項（`*`標識項）及其它未顯示列印參數相關項均會按照默認的 Profile 執行。
