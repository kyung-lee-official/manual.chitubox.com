# 列印機列表

點擊軟體左側導覽欄的「列印機」選項，開啟列印機列表介面。

如果尚未新增列印機，直接點擊「新增列印機」按鈕跳轉至<NextLink href="/zh-TW/chitu-manager/latest/sections/add-printers">新增設備</NextLink>介面。

<ImageContainer
	src={"/images/docs/zh-CN/chitu-manager/003-printer-list-main-printer.png"}
	style={{ maxWidth: "1050px" }}
/>

<Admonition type="note" title="注意">
	請確保列印機和電腦在同一局域網下。
</Admonition>

如果您已新增列印機，可以在此介面查看所有已新增的列印機、列印機的即時狀態，以及進行其他的一些簡單的<NextLink href="/zh-TW/chitu-manager/latest/sections/operations">操作</NextLink>。

## 基本狀態

### 攝像頭接入狀態

出現 <CameraStatus /> 表示列印機已接入攝像頭。

### U 盤接入狀態

出現 <UsbDriveStatus /> 表示列印機已接入 U 盤。

### 網路連線狀態

-   <WiredNetwork /> 表示列印機使用有線連線網路。

-   <WirelessNetwork /> 表示列印機使用無線連線網路。

### 設備即時狀態

您還可以在此介面查看設備的即時狀態，共有離線、待機、連線中、列印中、列印完成、已暫停等狀態。

## 詳細狀態

除了在列表中查看所有列印機的基本狀態，還能點擊單個設備卡片，開啟選中設備的詳細狀態面板。

<ImageContainer
	src={"/images/docs/zh-CN/chitu-manager/003-printer-list-detailed-list.png"}

### 離型膜使用次數

<ReleaseFilmUsageCount /> 表示離型膜的使用次數。

### 曝光屏使用時長

<ExposureScreenUsageTime /> 表示曝光屏的使用時長。
