## 插件要求

在 CHITUBOX 中，有些列印機是通過插件的形式支持的。因此，您需要在保存切片前下載並導入對應的插件文件。

| 列印機型號         | 插件                                                                                                                                                                                                 |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Longer 3D Orange10 | <Link href="/plugins/Longer3d-Orange10-Windows64bit-V1.4.CHplugin" download>Windows 版本</Link> <br /> <Link href="/plugins/Longer3d-Orange10-macOS-V1.4.CHplugin" download>Mac 版本</Link> |
| Longer 3D Orange30 | <Link href="/plugins/Longer3d-Orange30-Windows64bit-V1.4.CHplugin" download>Windows 版本</Link> <br /> <Link href="/plugins/Longer3d-Orange30-macOS-V1.4.CHplugin" download>Mac 版本</Link> |
| Longer 3D Orange4K | <Link href="/plugins/Longer3D-Orange4K-Windows64bit-V1.5.CHplugin" download>Windows 版本</Link> <br /> <Link href="/plugins/Longer3d-Orange4K-macOS-V1.5.CHplugin" download>Mac 版本</Link> |

下載完成后，您可以在 **左上角下拉菜單 > 幫助 > 插件** 打開插件窗口，然後導入插件文件。

## macOS

在 macOS 上使用 CHITUBOX Basic 無需安裝流程，因為它是以可攜式 (portable) 的形式提供的。您只需下載 CHITUBOX Basic 的最新版本，然後將其拖放到「應用程式」資料夾中即可。
