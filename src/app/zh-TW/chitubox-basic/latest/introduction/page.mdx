# 介紹

CHITUBOX Basic 是一款免費的 3D 列印切片軟件，可將 3D 網格檔案切片並轉換為可列印檔案。它旨在讓 3D 列印更輕鬆、更高效，簡化整個 3D 列印過程，讓用戶能夠簡單快速地將 3D 模型轉換為 3D 列印件。軟件支持多種 3D 列印機和配置檔案，並且提供了豐富的選項來優化 3D 列印模型。

## 新特性 ✨

### 全新的 UI 界面

<ThemeScrollSlider lang={"en"} />

很高興地宣布 CHITUBOX Basic 2 推出了全新的用戶界面。新的 UI 界面讓用戶體驗更輕鬆、更直觀。我們改進了軟件的導航和組織方式，使您更容易找到所需的工具和功能。我們還添加了一系列新功能，包括主題、改進的物件操作等等。使用新 UI，您的 3D 列印體驗將更快、更高效、更愉悅。

### 支持 DirectX 和 Metal

<DirectXAndMetal />

通過支援 DirectX 和 Metal，讓 CHITUBOX Basic 現在可以渲染更逼真、更複雜的 3D 模型，具有更豐富的細節和更逼真的照明、陰影和菲涅爾效果。這些技術還能提高效能，讓您充分發揮顯示卡的威力。現在在 Windows 和 macOS 平台上使用 CHITUBOX Basic 時，您可以體驗到最高級別的圖形效能。

### RMA

<Rma />

我們推出了新的 RMA（樹脂聯盟）功能。此功能使您可以輕鬆地為不同品牌的樹脂材料選擇合適的設定，它還支持雲儲存，因此您可以從任何地方訪問您的設定。此外，此功能會不斷地更新，因此您始終可以獲得最新配置。有了這個新功能，您現在可以快速輕鬆地設定列印機，以獲得最佳 3D 列印結果。

### 基本標準零件

<StandardParts />

此功能提供常用的基礎形狀零件庫，這樣您就不必花時間在互聯網上搜尋。
