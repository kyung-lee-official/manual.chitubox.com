# 設定支撐參數

點擊主視窗右上方的支撐圖示 <ChituboxSupport /> 即可進入支撐配置介面。

## Z 擡升高度

模型最低點與列印平台之間的高度距離。默認值為 5 mm。

## 支撐設定

<ImageContainer
	src={
		"/images/docs/zh-TW/chitubox-basic/2.x.x/010-configure-support-parameters-support-settings.png"

/>

您可以在此處 <ChituboxProfileImportIcon /> **導入**、<ChituboxProfileExportIcon /> **導出**和 <ChituboxProfileResetIcon /> **重設**支撐設定。

CHITUBOX 提供了三種預設的支撐設定：細、中、粗。如果這些設定不符合您的模型，您可以自訂支撐設定並導出以供日後使用。

## 自動支撐

自動支撐有兩種模式：**+全部** 和 **+平台**。

**+全部** 允许支撐在模型表面生成，而 **+平台** 只允许支撐在平台上生成。

## 操作模式

有三種操作模式：**添加**、**刪除** 和 **編輯**，快捷鍵分別是 **A**、**D** 和 **E**。三種模式均支援多選，您可以按住 Ctrl 鍵後點擊滑鼠左鍵並拖曳來選擇操作區域。

### 添加

<ChituboxAddSupportIcon /> 點击模型添加支撐。

新增支撐有**常規模式**和**直連模式**兩種，預設為常規模式。

直連模式允許您在模型和支撐主幹之間新增支撐。其規則如下：

-   如果您先點擊模型，您可以選擇將支撐另一端固定在模型上，或現有支撐的主幹上。
-   如果您先點擊支撐，您可以選擇將支撐另一端固定在現有支撐的主幹上。

如果您需要切換到直連模式，您需要先彈出支撐模式面板，

<ImageContainer
	className={"max-w-sm"}
	src={
		"/images/docs/zh-CN/chitubox-basic/2.x.x/010-configure-support-parameters-mode-panel.png"
	}
/>

然後選擇直連模式。

<ImageContainer
	className={"max-w-sm"}

### 刪除

<ChituboxDeleteSupportIcon /> 點擊選擇想要刪除的支撐，所選支撐將變為紅色，再次點擊按鈕或按下
Delete 鍵刪除支撐。

### 編輯

<ChituboxEditSupportIcon /> 點擊選擇想要編輯的支撐，所選支撐將變為綠色，按住
Ctrl 鍵可以選擇多個支撐，可以調整選中支撐的參數，也可以拖動支撐調整支撐位置。

編輯模式下，您可以雙擊支撐頂部的節點來新增節點支撐（按 Esc 鍵退出節點支撐）。

## 清除全部

點擊此按鈕，所有支撐都會被清除。
