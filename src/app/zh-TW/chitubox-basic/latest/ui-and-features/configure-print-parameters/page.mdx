# 設定列印參數

在設定列印參數之前，您需要先添加列印機。如果您還沒有添加列印機，點擊主視窗底部的機器管理欄的添加按鈕 <ChituboxAddMachine size={24} />，選擇列印機品牌展開可用列印機，選擇您要添加的列印機，點擊**下一步**按鈕添加列印機。

添加列印機後，您可以為列印機配置列印參數。一個列印機可以有多个配置檔案，您可以在配置檔案面板中管理您的配置檔案。

在設定參數面板，你可以 <ChituboxProfileDownloadIcon /> **下載設定參數** (材質庫或自訂), <ChituboxProfileUploadIcon /> **上傳所有設定參數** (自訂), <ChituboxProfileAddIcon /> **新增設定參數**, <ChituboxProfileImportIcon /> **匯入設定參數** (本地), <ChituboxProfileExportAllIcon /> **匯出所有設定參數** (本地).

將滑鼠懸浮在配置參數的圖示 <ChituboxProfileMenuIcon /> 上將彈出對於該配置的可用操作, 包括 <ChituboxProfileUploadIcon /> 上傳, <ChituboxProfileRenameIcon /> 重新命名, <ChituboxProfileExportIcon /> 匯出, <ChituboxProfileResetIcon /> 重置, <ChituboxProfileDeleteIcon /> 刪除。

## 機器

在這裡您可以配置列印機的基本信息，包括列印機的名稱、機器解析度、尺寸等。

### 名稱

列印機的名稱，可編輯，預設為列印機型號。

### 類型

列印機的型號，不可編輯。

### 鏡像

曝影像鏡像模式。在大多數情況下，您不需要更改此設定，除非您使用默認列印機類型（Other > default）。

### 分辨率和尺寸

列印螢幕的分辨率和尺寸。在大多數情況下，您不需要更改此設定，除非您使用默認的列印機類型（Other > default）。

連結/取消連結按鈕 <ChituboxProfileLinkIcon /> 允許您連結/取消連結尺寸和分辨率或 X 和 Y。任何連結的值都將在更改其他值時自動更新。

如果尺寸與分辨率不進行連結（分辨率保持不變），螢幕尺寸越小，則實際列印尺寸越大，反之亦然。

<Admonition type="warning" title="警告">
	分辨率必須與列印機的實際分辨率匹配，否則列印機可能會報錯。
</Admonition>

### 建造區域偏移

建造區域的偏移量。通過啟用此選項，您可以調整到列印螢幕邊緣的偏移量。偏移區域將不會被列印。如果您的螢幕邊緣有壞點，設定偏移可以避開壞點。

## 樹脂

在這裡您可以配置樹脂的基本信息，包括樹脂的名稱、類型、密度、單價等。CHITUBOX 將使用這些信息來計算樹脂的使用量和成本。

### TSMC（二段運動控制）

<Admonition type="tip">
	從 Anycubic Photon Workshop 而來？小心！

	* 由於 Anycubic Photon Workshop 提供基本和高級設置，而 CHITUBOX 的格式缺少專門的字段來區分它們，因此，對於不相容高級設置的印表機，CHITUBOX 僅顯示基本參數。相反，對於同時支援兩者的印表機，CHITUBOX 僅顯示進階參數。如果您的印表機支援進階設置，但您不想使用 TSMC 功能，您只需為兩個階段輸入相同的速度，並確保總升降距離與所需距離保持一致。

	* UI 中 TSMC 參數的順序不同。在 Photon Workshop 中，[0] 表示距離螢幕較近的運動階段，而 [1] 表示距離螢幕較遠的階段。然而，CHITUBOX 在版面配置輸入文字方塊時遵循運動順序。一開始這可能會令人困惑。
</Admonition>

你可能已經注意到列印設定中有兩組抬升和回縮參數，這裡是設定 TSMC 的地方。

當列印平台開始帶著模型升起時，樹脂料槽底部的離型膜會將剛固化的層向下拉，直到脫離，這稱為「吸力效應」。當與離型膜接觸面積較大，且列印平台以較高速度拉開時，吸力效應較為明顯。吸力效應會導致模型變形，甚至導致模型從列印平台上脫落。我們建議保持緩慢的升起速度，直到模型脫離離型膜。當吸力效應不再是問題時，隨後加快升起速度可以提高列印效率。收回階段的原理類似。

## 高級

### 底層光强度 PWM

底層光的 <SpanA href="https://en.wikipedia.org/wiki/Pulse-width_modulation">PWM</SpanA>，範圍從 `0` 到 `255` 或從 `0%` 到 `100%`，控制底層的 UV 光源功率。

### 光强度 PWM

普通層光的 <SpanA href="https://en.wikipedia.org/wiki/Pulse-width_modulation">PWM</SpanA>，範圍從 `0` 到 `255` 或從 `0%` 到 `100%`，控制普通層的 UV 光源功率。

### 圖片灰階

類似於燈光 PWM，範圍從 `0` 到 `255`，但是圖像灰階不是控制 UV 光源的功率，而是控制 LCD 屏幕的透光率。值越大，透光率越高。

### 抗鋸齒

有兩種抗鋸齒模式：**灰階** 和 **抗鋸齒等級（舊）**。

#### 灰階

該模式適用於 _.ctb_ 格式。

| 參數         | 描述                                                                                           |
| ------------ | ---------------------------------------------------------------------------------------------- |
| 灰階         | 範圍：`0` ~ `255`。`0` 代表黑色，`255` 代表白色。它決定了可用于抗鋸齒的灰階範圍。              |
| 影像模糊像素 | 可用值：`2` ~ `8`。決定邊緣過渡的像素數。值越高，邊緣就越柔和。類似於 Photoshop 中的羽化功能。 |

#### 抗鋸齒等級

這是舊模式，只適用於 _.cbddlp_ 格式。

| 參數       | 描述                                                                                                                |
| ---------- | ------------------------------------------------------------------------------------------------------------------- |
| 抗鋸齒等級 | 可用值：`2`、`4`、`8`。它決定了邊緣的曝光時間。值越高，曝光時間就越長，這將導致邊緣更加銳利，從而降低抗鋸齒的效果。 |

### 收縮補償

模型在 X，Y，Z 方向上按照一定比例進行收縮補償。百分比是與原始模型相比的大小，默认值為 100%。

百分比數值越大，模型實體越大。

百分比數值越小，模型實體越小。

### 公差補償

普通層內外直徑補償。

當 a 值較大時，內直徑較短，模型體積較大。

當 a 值較小時，內直徑較長，模型體積較小。

當 b 值較大時，外直徑較長，模型體積較大。

當 b 值較小時，外直徑較短，模型體積較小。

打開 <SpanA href="https://www.bilibili.com/video/BV1tw411R7QB?share_source=copy_web">📺 這條影片</SpanA> 和 <SpanA href="https://www.chitubox.com/zh-Hans/article/support/howto/chitubox-free/settings/how-to-use-tolerance-compensation-on-chitubox">📃 這篇文章</SpanA> 了解更多關於**公差補償**的內容。

### 底層公差補償

底層內外直徑補償。通常此設置用於解決“大象腿”問題。

當 a 值較大時，內直徑較短，模型體積較大。

當 a 值較小時，內直徑較長，模型體積較小。

當 b 值較大時，外直徑較長，模型體積較大。

當 b 值較小時，外直徑較短，模型體積較小。

### 層列印時間補償

一些高解析度列印機在加載數據時可能會產生額外的延遲，這將導致列印時間預估不准確。您可以設置時間補償來改善這個問題。

-   手動輸入

    每層列印時間補償：一個 2 位數，範圍在 `-99.99` 到 `99.99` 之間，默認為 `0` 秒。
    當啟用時間補償時，單層預計時間 = 曝光時間 + 等待時間 + 移動時間 + 每層列印時間補償
