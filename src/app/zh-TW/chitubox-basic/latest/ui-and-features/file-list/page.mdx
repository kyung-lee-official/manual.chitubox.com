# 檔案列表

所有已載入完成的檔案都會顯示在檔案列表中。您可以同時選擇多個檔案，也能透過單擊全選按鈕來選擇所有檔案。

<ImageContainer
	src={"/images/docs/zh-TW/chitubox-basic/2.x.x/007-file-list-panel.png"}
/>

點擊複選框圖示 <ChituboxCheckboxIcon /> 可以選擇或取消選擇該檔案。

點擊眼睛圖示 <ChituboxEyeIcon /> 可以隱藏或顯示該檔案。點擊添加圖示可以添加檔案到檔案列表。

點擊標尺圖示 <ChituboxRulerIcon /> 可以顯示所有檔案的<Link href={"https://zh.wikipedia.org/zh-tw/%E5%8C%85%E5%9B%B4%E4%BD%93"}>包圍體</Link>尺寸。

點擊垃圾桶圖示 <ChituboxDeleteIcon /> 可以刪除所選檔案。
