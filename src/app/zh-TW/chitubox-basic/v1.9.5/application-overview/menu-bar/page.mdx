## 修復

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/006-application-overview-menubar005.png"
	}
/>

<ImageContainer
	src={

修復面板上的下拉菜單可以選擇**反轉法線**或者**自動修復**。

### 反轉法線

將模型中所有面的法線進行翻轉。

### 自動修復

自動檢測並修復模型上的瑕疵例如：法線翻轉、破損等。
