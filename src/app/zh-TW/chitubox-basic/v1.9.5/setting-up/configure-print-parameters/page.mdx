##### 2.2 燈滅延遲

###### 2.2.1 什麽是燈滅延遲

**燈滅延遲** 指的是在 3D 列印過程中，列印一層時在列印平台上 **抬升**，**等待** 和 **回程** 所花費的總時間。

在開始列印下一層之前，確保樹脂有充足的穩定固化時間是非常重要的。大多數情況下，燈滅延遲的時間設定得越久，得到的列印結果就越好。但是如果你設定得太久，就會耗費很多時間成本。

**抬升** 和 **回程** 由抬升距離決定，而 **等待** 則不同，一般情況下我們推薦 2 到 3 秒。

###### CHITUBOX 中的燈滅延遲計算

為了計算燈滅延遲，你需要知道抬升距離，抬升速度以及回程速度。算出抬升和回程的總時間，再加上額外 2 到 3 秒的等待時間，你就得到了燈滅延遲時間。

燈滅延遲 = 抬升時間 + 回程時間 + 等待時間

舉個例子：

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-configure-printer-parameters003.2.png"
	}
/>

這裏 抬升距離 是 7mm，抬升速度 是 70mm/min (1.167mm/sec)， 因此抬升時間 $T_L$ 為：

$$T_L = \frac{7}{1.167} = 5.998\ \text{秒}$$

類似地，回程速度 是 210mm/min (3.5mm/sec), 回程時間 $T_R$ 是

$$T_R = \frac{7}{3.5} = 2\ \text{秒}$$

如果我們希望最小等待時間比 2s 長，那麽燈滅延遲 $T_{LOD}$ 則應該為：

$$T_{LOD} \geq 5.998 + 2 + 2 \approx 10\ \text{秒}$$

如果你在燈滅延遲中輸入一個小於 $ { T_L + T_R} $ 的數，則燈滅延遲不工作。

#### 3. 抗鋸齒

對圖像邊緣的三角形鋸齒進行柔化處理以實現物體更光滑的表面效果。相鄰三角形鋸齒間通過增加或刪除一定灰度的像素單元可以調控邊緣曝光的連續性和透光強度，進而實現邊緣區域更加光滑的效果。

-   _抗鋸齒級別_：`2`，`4`，`8`，**適用於 _.cbddlp_ 格式**，控制邊緣曝光時間。等級越高時間越長，邊緣越清晰，抗鋸齒效果越低。

-   _灰度級別_：`0` ~ `8`，**適用於 _.ctb_ 格式**，控制可用於抗鋸齒的灰度數量。將 `0`~`255` (`0` 代表黑色，`255` 代表白色) 的灰度範圍劃分為 8 個級別，級別越高，範圍越窄，越接近 `255` (白色)。

-   _圖像模糊_：啟用/關閉圖像模糊。**適用於 _.ctb_ 格式**。

    -   _圖像模糊像素_：`2` ~ `8`，決定了邊緣過渡像素的數量，等級越高，邊緣越模糊。類似於 Photoshop 中的羽化功能。

a 的值越小，內徑越大，模型實體越小。<br />
b 為正值（補償模型收縮部分），內徑減小，模型實體增大。<br />
b 為負值（補償模型膨脹部分），內徑增大，模型實體減小。

<Admonition type="note" title="注意">
	添加禁用底部公差補償時，普通層公差補償對全模型生效。
</Admonition>

#### 7. 列印時間補償

部分高解析度列印機在載入數據時可能會產生額外的延遲，從而導致預估列印時間不準確。可以通過設定列印時間補償來改善這個問題。

-   **手動輸入**

    層列印時間補償: 一個在 `-99.99` 和 `99.99` 之間的 2 位數，預設為 `0` 秒。

    當啟用時，

    單層的預估列印時間 = 曝光時間 + 等待時間 + 運動時間 + 層列印時間補償

-   **自動生成**

<ImageContainer
	src={
		"/images/docs/zh-CN/chitubox-basic/1.x.x/004-setting-up-configure-printer-parameters004-1.png"
	}
/>

$$\text{層列印時間補償} = \frac{\text{實際列印時間} - \text{軟件預測時間}}{\text{列印總層數}}$$

<Admonition type="note" title="注意">
	設定完參數請點擊<b>應用</b>，否則參數不生效。
</Admonition>
