# 要求

## 系統要求

### Windows

|                | 最低配置                                     | 推荐配置                                  |
| -------------- | -------------------------------------------- | ----------------------------------------- |
| 中央處理器     | Intel® Core™ i5 6600K 或者 AMD Ryzen™ 5 1600 | Intel Core i7 4790 或者 AMD Ryzen 7 2700X |
| 作業系統       | Windows 7 (64 位)                            | Windows 10 (64 位)                        |
| 隨機存取記憶體 | 16GB                                         | ≥ 32GB                                    |
| 硬碟           | 不低於 10GB 的硬碟存儲空間                   | 20GB 以上的固態硬碟                       |
| 圖形處理器     | GPU 隨機存取記憶體：≥1GB                     | GPU 隨機存取記憶體：≥1GB                  |
| 顯示解析度     | 解析度 ≥ 1280x960                            | 解析度 ≥ 1280x1024                        |

<Admonition type="warning" title="警告">
	部分 Quadro 系列圖形處理器可能與 CHITUBOX
	不兼容，如果您遇到此問題，可以將您的圖形處理器型號發送至電子郵箱 📧
	support@chitubox.com，這可能有助於我們分析問題。
</Admonition>

### macOS

|                | 最低配置                    | 推荐配置                         |
| -------------- | --------------------------- | -------------------------------- |
| 中央處理器     | Intel® 或 M1 中央處理器     | Intel® 或 M1 中央處理器          |
| 作業系統       | macOS Catalina (10.15 版本) | macOS Big Sur (11.6 版本) 或以上 |
| 隨機存取記憶體 | 16GB                        | ≥ 32GB                           |

<Admonition type="note" title="注意">
	理論上，您可以使用 Rosetta 2 在 M1 中央處理器的 Mac 上運行
	CHITUBOX（確保您的 Mac 版本已更新到 Big Sur）。
	但是，我们不能保證以這種方式運行 CHITUBOX 時沒有性能問題。
</Admonition>

### Linux

|                | 最低配置                                           | 推荐配置                         |
| -------------- | -------------------------------------------------- | -------------------------------- |
| 中央處理器     | 工作主頻不低於 2.0GHz 的 Intel® 或 AMD® 中央處理器 | 工作主頻不低於 3GHz 的多核處理器 |
| 作業系統       | Ubuntu 20.04                                       | Ubuntu 20.04 或更高版本          |
| 隨機存取記憶體 | 16GB                                               | ≥ 32GB                           |
| 硬碟           | 不低於 10GB 的硬碟存儲空間                         | 20GB 以上的固態硬碟              |
