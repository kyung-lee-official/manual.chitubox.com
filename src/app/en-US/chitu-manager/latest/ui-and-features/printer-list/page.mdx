# Printer List

Click the "Printer" option in the left navigation bar to open the printer list interface.

If you have not added a printer yet, click the "Add Printer" button to jump to the <Link href="/en-US/chitu-manager/latest/ui-and-features/management">Management</Link> interface.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/003-printer-list-main-printer.png"}
/>

<Admonition type="note">
	Please make sure the printer and the computer are on the same LAN.
</Admonition>

You can view all the added printers and their real-time status, and perform some other simple <Link href="/en-US/chitu-manager/latest/ui-and-features/operations">operations</Link> on this interface.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/003-printer-list-printer-list.png"}
/>

## Basic Status

### Camera Status

The <CameraStatus /> icon indicates that the printer is connected to a camera.

### USB Drive Status

The <UsbDriveStatus /> icon indicates that the printer is connected to a USB drive.

### Network Status

-   <WiredNetwork /> indicates that the printer is connected to the network
    using a wired connection.

-   <WirelessNetwork /> indicates that the printer is connected to the network
    using a wireless connection.

### Device Real-time Status

On this interface, you can also view the real-time status of the device, including offline, standby, connecting, printing, printing completed, paused, and other statuses.

## Detailed Status

You can also click on an individual device to open the detailed status panel for the selected device.

### Release Film Usage Count

<ReleaseFilmUsageCount /> indicates the number of times the release film has
been used.

### Exposure Screen Usage Time

<ExposureScreenUsageTime /> indicates the usage time of the exposure screen.
