# Introduction

ChituManager is a 3D printer management software designed to work with CHITUBOX slicing software, providing users with a comprehensive solution for managing 3D printers.  With ChituManager, you can view the status of your printers in real time, transfer files remotely, and initiate prints without having to be physically present at the printer. ChituManager also supports error reporting, historical record summarization and viewing, video monitoring and other features that simplify the 3D printing process, helping users save time and improve efficiency.

## Installation

Click the ChituManager icon at the top right corner of the CHITUBOX slicer's main interface. Alternatively, the software will display a download window and prompt you to download ChituManager before proceeding when you use the Network Sending function.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/001-installation-windows-step1.png"}
	style={{ maxWidth: "600px" }}
/>

Click "Download and Install".

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/001-installation-windows-step2.png"}
	style={{ maxWidth: "600px" }}
/>

Wait for the download to finish.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/001-installation-windows-step3.png"}
	style={{ maxWidth: "600px" }}
/>

Once the download is complete, the installation process will automatically start. Click "Next" to proceed.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/001-installation-windows-step4.png"}
	style={{ maxWidth: "600px" }}
/>

Select the installation path, we recommend you to use the default path, and click "Next" to continue.

<ImageContainer
	src={"/images/docs/en-US/chitu-manager/001-installation-windows-step5.png"}
	style={{ maxWidth: "600px" }}
/>

Wait for the installation to complete.
