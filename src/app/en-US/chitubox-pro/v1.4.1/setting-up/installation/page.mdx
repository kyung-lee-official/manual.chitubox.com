# Installation

## Windows

Double click CHITUBOX installer to start installing.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation001.png"
	}
/>

Click **Next**.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation002.png"
	}
/>

Agree the EULA.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation003.png"
	}
/>

Select your path for installation, and click **Install**.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation004.png"
	}
/>

Wait for the progress bar until it is done.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation005.png"
	}
/>

Click **Yes**, restart the resource manager to continue.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation006.png"
	}
/>

Click **Finish**, you're done!

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/004-setting-up-installation007.png"
	}
/>

<Admonition type="note">
	If you found issues that were claimed fixed in the update log but still exist after updating CHITUBOX Pro, please clean the cache and try again. Click <Link href="/en-US/faq/latest/chitubox-pro-faq#clean-the-cache">here</Link> to learn more about cache cleaning.
</Admonition>
