### Top

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-support004.top.png"
	}
/>

| Parameter             | Explanation                                                            |
| --------------------- | ---------------------------------------------------------------------- |
| Contact Shape         | Shape of the support tip contacting the model.                |
| Contact Diameter(mm)  | Diameter dimension of the support tip contacting the model |
| Contact Depth(mm)     | Depth dimension of the support tip contacting the model    |
| Connection Shape      | Shape of the support's top segment.                              |
| Upper Diameter(mm)    | Diameter of the support's top segment.                  |
| Lower Diameter(mm)    | Diameter of the support's top segment.                  |
| Connection Length(mm) | Length of the support's top segment.                                            |

### Middle

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-support004.mid.png"
	}
/>

| Parameter          | Explanation                                                                   |
| ------------------ | ----------------------------------------------------------------------------- |
| Shape              | Shape of the main support segment.                            |
| Upper Diameter(mm) | The upper end diameter(mm) of the main support segment.                           |
| Lower Diameter(mm) | The lower ebd diameter(mm) of the main support segment.                            |
| Angle(°)           | Maximum angle between the main support segment and the top support segment.          |
| Small Pillar Shape | Shape of small pillars' tips contacting the model. |
| Diameter(mm)       | Diameter of small pillars.                              |
| Upper Depth(mm)    | Depth of the small pillar's upper end extending into the model.               |
| Lower Depth(mm)    | Depth of the small pillar's lower end extending into the model.       |

### Bottom

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-support004.bot.png"
	}
/>

| Parameter            | Explanation                                                                                                                              |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Platform Touch Shape | Shape of the support's base contacting the printing platform.                                                                             |
| Touch Diameter(mm)   |Dimension of the support's base contacting the printing platform.                                                                     |
| Thickness            | Thickness of the support's base contacting the printing platform.                                                              |
| Model Contact Shape  | Shape of the connecting segment when the model is in direct contact with the support's base.            |
| Contact Diameter(mm) | Diameter of the connecting segment when the model is in direct contact with the support's base.                |
| Contact Depth(mm)    | Diameter of the connecting segment extending into the model when the model is in direct contact with the support's base.            |
| Contact Point        | The number of contact points when the model is in direct contact with the support's base. |

### Raft

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-support004.raf.png"
	}
/>

| Parameter            | Explanation                                                                                                                                                                  |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Raft Shape           | Shape of the supporting raft on the printing platform                                                                                                     |
| Raft Area Ratio(%)   | Controls the size of the supporting raft, by setting the ratio of the total area of the raft relative to the total area projected by the model to the printing platform |
| Raft Thickness(mm)   | Wall thickness of the supporting raft.                                                                                                                                           |
| Raft Height(mm)      | Overall height of the supporting raft.                                                                                                                                           |
| Raft Slope(°)        | Angle between raft's side wall and the printing platform                                                                                                                  |
| Grid Side Length(mm) | Width of each grid for grid type supporting raft.                                                                                                                                                         |
| Grid Width(mm)       | Spacing between each grid for grid type supporting raft.                                                                                                                                                           |

#### Enhanced Automatic Support

Auxillary supports with cross bracings will be added automatically to certain isolated supporting columns during Auto Support.

-   Triangular Reinforcement (TR): The program will try to automatically connect adjacent support columns to form stable triangular structures with Trianglular Reinforcement during Auto Support.

-   Global Reinforcement (GR): The program will try to inter-connect all adjacent support columns to form a more stable structure composed of small triangular structures under Global Reinforcement during Auto Support.
