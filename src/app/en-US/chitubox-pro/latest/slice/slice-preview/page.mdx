# Slice Preview

The main window will be split into model view and slice preview when the slice process is done.**Model View** displays sliced models, and **Slice Preview** shows a preview of every sliced layer.

## Information Preview

### Basic Information Preview

You can check basic print information on the top right estimation panel. Print information includes machine model, resin, volume, weight, cost, and estimated print time.

<Admonition type="note" title="Note">
	The volume of resin is a simple calculation of summing up of all voxels (area of pixels x layer height).
	This is, however, a theoretic value, that can be affected by many other factores in reality, such as residual resin on the build plate or the model, resin volatility, density change during curing etc.
</Admonition>

## Slice Editor

### Slice Parameter Editor

<SliceParameterEditor /> Edit and adjust printer parameters of the sliced models.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/016-slice-preview-slice-parameter-editor.png"

This window provides quick access to some parameters found under **Slice Settings > Print**, allowing users to change these parameters without quiting the slice preview window.

<Admonition type="note" title="Note">
	**Slicing Parameter Editing** is only available under **Single Parameter Slice** mode.
</Admonition>

### Edit Pixels

<EditPixels /> adds or deletes pixels in the current slice file.

#### Add Pixels

1. Left click in the slice preview window to add pixels.

1. Hold Ctrl and drag Left-button of the mouse to add continuous pixels. 

#### Delete Pixels

1. Right click in the slice preview window to delete pixels.

1. Hold Ctrl and drag Right-button of the mouse to erase pixels continously.

<Admonition type="tip" title="Note">
	Hold Ctrl and scroll the Middle-button of the mouse to resize the pen
</Admonition>

Click the **Save Slice** button to save the slice file.

<Admonition type="note" title="Note">
	* Only file formats supported by the current printer are displayed in the save file dialog. Please go back and check if the right printer is seleted if no appropriate file format is available.
    * The **Default** printer allows you to save the slice file to any format supported by CHITUBOX. You should choose the **Default** printer type when no configuration file for your printer is loaded into CHITUBOX.
</Admonition>

The **Network Sending** button is only available when your printer supports the function. Network Sending allows direct sending of print data to the machine for quick printing.
