## Windows

Download the installation package from our <Link href="https://www.chitubox.com/en/page/chitubox-pro">official site</Link> and double-click the downloaded installer to start the installation process.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/002-installation-windows-step1.png"
	}
	style={{ maxWidth: "600px" }}
/>

Click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/002-installation-windows-step2.png"
	}
	style={{ maxWidth: "600px" }}
/>

Read the End User License Agreement. Check the "I accept the license" option if you agree to the terms, and click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/002-installation-windows-step3.png"
	}
	style={{ maxWidth: "600px" }}
/>

Choose the installation path. We recommend you using the default path. Click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/002-installation-windows-step4.png"
	}
	style={{ maxWidth: "600px" }}
/>

Wait for the installation to complete.
