| File Type | File Suffix | Description                                                 |
| -------- | ---------- | ---------------------------------------------------- |

| Configuration File | \*.cfg     | Includes printer configuration and support configuration data                     |

## File Naming Settings

### Model File

<Entry mainMenu={["Settings", "File", "Model Rename"]} />

Automatically adds suffix to the model file name based on your presets. i.e."#number", "Hollowed", "Repaired" or "Mirrored".

### Project File

<Entry mainMenu={["Settings", "File", "Project File Naming"]} />

-   Append Model Filename Sequentially: generate the filename based on the order in which the models are loaded into the file.

### Slice File

<Entry mainMenu={["Settings", "File", "Slice File Naming"]} />

Configure naming rules for slice files.

### Name Replacement

#### Duplicate Name Replacement

<Entry mainMenu={["Settings", "File", "Duplicate Replacement"]} />

Set the rules for conflicting file names.

#### Character Replacement

<Entry mainMenu={["Settings", "File", "Character Replacement"]} />

Certain operating systems do not allow special characters in file names. These characters will be automatically replaced based on preset rules.

#### Custom Character Replacement

<Entry mainMenu={["Settings", "File", "Custom Character Replacement"]} />

Allows you to customize character replacement rules.
