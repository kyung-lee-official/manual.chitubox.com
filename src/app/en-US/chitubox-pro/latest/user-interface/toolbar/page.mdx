## Prepare, Analyze/Measure Interface

The toolbar in the Prepare and Analyze/Measure interface contains six tools: <Link href="/en-US/chitubox-pro/latest/chitu-action">ChituAction</Link>, <Link href="/en-US/chitubox-pro/latest/model-processing#select">Selection</Link>, <Link href="/en-US/chitubox-pro/latest/model-processing#move">Move</Link>, <Link href="/en-US/chitubox-pro/latest/model-processing#rotate">Rotate</Link>, <Link href="/en-US/chitubox-pro/latest/model-processing#scale">Scale</Link> and <Link href="/en-US/chitubox-pro/latest/model-processing#mirror">Mirror</Link>.

The toolbar in the Support interface includes <Link href="/en-US/chitubox-pro/latest/support/magic-support">Magic Support</Link>, <Link href="#support-group">Support Group</Link> and <Link href="#selection-tools">Selection Tools</Link>.

### Support Group

Group your supports by model and type for easier management. This makes it easy to preview all support details quickly and batch edit or delete supports.

<ImageContainer
	src={"/images/docs/en-US/chitubox-pro/2.x.x/toolbar-support-group.png"}
/>

### Selection Tools

You can use different tools to select areas of your model for support addition, or to select added supports for editing or deletion. Five selection tools are available: Brush, Line, Rectangle, Polygon, and Free Curve.
