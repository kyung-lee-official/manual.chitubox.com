# Slice Panel

The slice panel is located in the bottom right corner of the main interface. It includes slice settings, slice modes, and single/multi-parameter slicing buttons.

## Slice Settings

Open the <Link href="/en-US/chitubox-pro/latest/printer-management/printer-configuration">slice settings</Link> for the currently selected printer.

## Slice Mode

Choose between single parameter or multi-parameter <Link href="/en-US/chitubox-pro/latest/slice/slice-mode">slice mode</Link>. If the selected printer does not support multi-parameter slicing, this option will be unavailable.

## Single/Multi-Parameter Slicing

Click this button to slice the project and enter the <Link href="/en-US/chitubox-pro/latest/slice/slice-preview">slice preview</Link> interface.
