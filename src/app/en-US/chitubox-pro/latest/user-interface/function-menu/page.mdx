# Function Menu

<ImageContainer src="/images/docs/en-US/chitubox-pro/2.x.x/function-menu.png" />

## Prepare Function Tab

Click "Prepare" on the Function Menu to switch to the Prepare function tab.

<ImageContainer src="/images/docs/en-US/chitubox-pro/2.x.x/prepare-function-panel.png" />

### Add Machine

<Link href="/en-US/chitubox-pro/latest/printer-management/add-and-delete-printers#add-printers">Add a New Machine</Link>.

### History

<Link href="/en-US/chitubox-pro/latest/file#open-recent-files">Visit Recently Opened Files</Link>.

### Open

Open project files, import 3D models and slice files.

### Export

Export all models, export selected models in combination or separately (including supports). 

Files can be exported as project files(.chitubox/.ctp) or model files.

### Save

<Link href="/en-US/chitubox-pro/latest/file#save-project">Save the current project file</Link>.

### Copy

<Link href="/en-US/chitubox-pro/latest/model-processing#copy">Copy</Link> one or multiple selected models.

### Split Model

<Link href="/en-US/chitubox-pro/latest/model-processing#split-model">Split an originally assembled model</Link> into individual parts.

### Auto Layout

<Link href="/en-US/chitubox-pro/latest/model-processing#auto-layout">Quickly arrange and layout </Link> multiple models based on selected options.

### Add Tag

<Link href="/en-US/chitubox-pro/latest/model-processing#add-tag">Add preset tags</Link> to models.

### Boolean Operation

Carry out <Link href="/en-US/chitubox-pro/latest/model-processing#boolean-operation">Boolean Operations</Link> such as **Join**, **Intersect** or **Subtraction** for multiple selected models.

### Hollow

<Link href="/en-US/chitubox-pro/latest/model-processing#hollow">Hollow out</Link> selected models.

### Hole

<Link href="/en-US/chitubox-pro/latest/model-processing#hole">Drill Holes</Link> on selected models.

## Support Function Tab

Click "Support" on the Function Menu to switch to the Support function tab.

<ImageContainer src="/images/docs/en-US/chitubox-pro/2.x.x/support-function-panel.png" />

### Symmetrical Support

Add <Link href="/en-US/chitubox-pro/latest/support/add-edit-delete#symmetrical-support">Symmetrical Supports</Link> to models.

### Merge Supports

<Link href="/en-US/chitubox-pro/latest/support/add-edit-delete#merge-supports">Automatically merge some or all supports</Link>. 

### Overhang Detection

<Link href="/en-US/chitubox-pro/latest/risk-detection#overhang-detection">Detect Overhang areas</Link> and add supports.
