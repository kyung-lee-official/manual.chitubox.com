# Slice

## Single Parameter and Multi-parameter

Slice is the final step before exporting a printable file.

Choose **Single Parameter Slice** or **Multi-parameter Slice** and then click the slice button to enter slicing preview.

### Single Parameter Slice

Uses the default profile from the slicing settings list for all model data within the printable range on the platform.

### Multi-parameter Slice

<Admonition type="note">
	Only supported by some printers.
</Admonition>

**Multi-parameter Slice** provides 3 modes, **Model**, **Height**, and **Cross-sectional Area**.

#### 1. Model

Different models use different printing parameters. <br />

<ImageContainer
    src={
    	"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-slice-preview017.png"
    }

/>

1. Models without specified Profile comply with the Profile use the **Default Profile**.

    <ImageContainer
    	src={
    		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-slice-preview017.1.1.png"
    	}
    />

1. Changes to profiles with the same name in the same printer environment will be synchronized.

1. Uneditable items `*` and other non-displayed printing parameters will follow the default profile.

    <ImageContainer
    	src={
    		"/images/docs/en-US/chitubox-pro/1.x.x/006-application-overview-slice-preview017.1.png"
    	}
    />

#### 3. Cross-sectional Area

Different cross section areas use different parameters. The percentage refers to the the cross-section relative to the total effective area of the platform.

<ImageContainer
	src={

1. Sections without specified Profile use the **Default Profile**.

1. Parameters for bottom layers and transition layers follow the **Default Profile**.

1. Uneditable items `*` and other non-displayed printing parameters will follow the default profile.

    <ImageContainer
