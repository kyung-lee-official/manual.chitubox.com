The Repair tab includes many functions for inspecting and fixing 3D model erros to improve success rates in Printing.

## Error List

The Error List on the right displays all detected model errors. Supported error types include: **Excess Shells**, **Holes**, **Bad Side**, **Reverse** and **Intersect**.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-pro/2.x.x/repair--error-list.png"
	}
/>

Click on the **Detect** button to run automatic error detection and the software will display model error types and related data. Choose Basic Repair or Advanced Repair to fix most error types with a single click.

## Toolbar

The **Toolbar** on the left includes multiple tools for quick selecting of model areas that need repair. The quick selection tools include **Brush**, **Line**, **Rectangular**, **Polygon** and **Free Box**.

	src={
		"/images/docs/zh-CN/chitubox-pro/2.x.x/003-repair.png"
