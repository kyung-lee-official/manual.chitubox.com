Click **Apply** to create supports based on the order of support settings from top to bottom.

## Manage Magic Support Profiles

In the Profile Panel you can <ChituboxProfileAddIcon /> add new profiles, <ChituboxProfileDownloadIcon /> download profiles, <ChituboxProfileImportIcon /> import profiles, <ProfileExportAllIcon /> export all Magic Support profiles.

By hovering <ChituboxProfileMenuIcon /> icon, you can see the available operations for the configuration, including <ChituboxProfileUploadIcon /> upload, <ChituboxProfileRenameIcon /> rename, <ChituboxProfileExportIcon /> export, <ChituboxProfileResetIcon /> reset, <ChituboxProfileDeleteIcon /> delete.

## Workflow Management

### Add/Delete a Step

-   Add a step: click the **+** sign at the bottom of a step card to add a new auto support step below.

-   Delete a step: click the **-** sign at the right side of a step to delete an unnecessary step.

### Step Settings

| Parameters        | Description                                     |
| ----------------- | ----------------------------------------------- |
| Support Parameter | Choose a support parameter file                 |
| Auto Support Mode | Choose a mode for how auto support is generated |
| Auto Support Type | Choose the type of auto support generated       |

### Merge Supports

When enabled, added supports will be automatically merged at the end.
