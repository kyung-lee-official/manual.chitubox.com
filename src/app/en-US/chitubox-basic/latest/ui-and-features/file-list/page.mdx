# File List

All loaded files are displayed in the file list. You can select multiple files at the same time or select all files by clicking the select all button.

<ImageContainer
	src={"/images/docs/en-US/chitubox-basic/2.x.x/007-file-list-panel.png"}
/>

Click the checkbox icon <ChituboxCheckboxIcon /> to select/deselect the file.

Click the eye icon <ChituboxEyeIcon /> display/hide the file.

Click the add icon <ChituboxAddFileIcon /> to add files.

Click the ruler icon <ChituboxRulerIcon /> to display the the <Link href={"https://computersciencewiki.org/index.php/Bounding_boxes"}>bounding box</Link> dimension of all files.

Click the trash icon <ChituboxDeleteIcon /> to delete the selected files.
