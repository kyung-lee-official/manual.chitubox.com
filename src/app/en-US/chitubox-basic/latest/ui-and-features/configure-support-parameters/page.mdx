# Configure Support Parameters

Click <ChituboxSupport /> to switch to the support view.

## Z Lift Height

The height distance between the lowest point of the model and the platform. The default value is 5mm.

## Support Settings

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/2.x.x/gif/SupportSettings.png"
	}
/>

Here you can <ChituboxProfileImportIcon /> **import**, <ChituboxProfileExportIcon /> **export**, and <ChituboxProfileResetIcon /> **reset** support settings.

CHITUBOX Provides 3 default support settings: Light, Medium, and Heavy. You can also create customized support settings and export them for future use.

## Auto Support

There are 2 modes for auto support: **+All** and **+Platform**.

**+All** allows supports to be generated based on all model surfaces, while **+Platform** only allows supports to be generated on the platform.

## Manual Support Operation

Manual Support Operation allows: **Add**, **Delete**, and **Edit** of supports, with corresponding shortcut keys "A", "D", and "E". Hold down the Ctrl key and drag the mouse to select all supports within the frame and batch **Add**, **Delete** and **Edit** supports.

### Add

<ChituboxAddSupportIcon /> Click the model to add supports.

There are two modes for adding supports: **Regular** (default) and **Connection**.

The Connection mode allows you to add supports between models and existing support trunks, when complying to the following rules:

-   After first clicking on a model surface, you can click and attach the other end of the support on the same model's surface or an existing support trunk.
-   After first clicking on a support trunk, you can click and attach the other side of the support on another existing support trunk.

In order to use Connection model, pop out the support mode panel first:

<ImageContainer
	className={"max-w-sm"}
	src={

Then select Connection mode on the panel:

### Delete

<ChituboxDeleteSupportIcon /> Click supports to highlight target supports in red, and then click the button again or press Delete key to delete selected supports.

### Edit

<ChituboxEditSupportIcon /> Click supports to highlight target supports in
green, or hold down Ctrl key to select multiple supports. You can then adjust parameters of the selected supports. You can also drag supports to adjust their positions.

Under Edit mode, double click a support node to generate support segments originated from the note (press Esc to exit).

## Clear All

Delete all supports by clicking this button.
