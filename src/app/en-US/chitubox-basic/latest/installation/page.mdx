## Plugin Requirements

Some printers are supported in CHITUBOX through plugins. Therefore, you need to download and import the corresponding plugin files before saving sliced files.

| Printer Model      | Plugin                                                                                                                                                                                               |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Longer 3D Orange10 | <Link href="/plugins/Longer3d-Orange10-Windows64bit-V1.4.CHplugin" download>Windows Version</Link> <br /> <Link href="/plugins/Longer3d-Orange10-macOS-V1.4.CHplugin" download>Mac Version</Link> |
| Longer 3D Orange30 | <Link href="/plugins/Longer3d-Orange30-Windows64bit-V1.4.CHplugin" download>Windows Version</Link> <br /> <Link href="/plugins/Longer3d-Orange30-macOS-V1.4.CHplugin" download>Mac Version</Link> |
| Longer 3D Orange4K | <Link href="/plugins/Longer3D-Orange4K-Windows64bit-V1.5.CHplugin" download>Windows Version</Link> <br /> <Link href="/plugins/Longer3D-Orange4K-macOS-V1.5.CHplugin" download>Mac Version</Link> |

Once the download is complete, you can open the plugin window by going to the **drop-down menu at the upper left corner > Help > Plugins**, and then import the plugin file.

## macOS

CHITUBOX Basic for macOS doesn't require an installation process since it is a portable application. Simply download the latest version of CHITUBOX Basic and drag and drop the CHITUBOX icon to the "Applications" folder.
