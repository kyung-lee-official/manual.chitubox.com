## Plugin Requirements

Longer 3D printers are supported in CHITUBOX through plugins. Therefore, you'll need to download and import the appropriate plugin file before saving sliced files.

| Printer Model      | Plugin                                                                                                                                                                                               |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Longer 3D Orange10 | <SpanA href="/plugins/Longer3d-Orange10-Windows64bit-V1.4.CHplugin" download>Windows Version</SpanA> <br /> <SpanA href="/plugins/Longer3d-Orange10-macOS-V1.4.CHplugin" download>Mac Version</SpanA> |
| Longer 3D Orange30 | <SpanA href="/plugins/Longer3d-Orange30-Windows64bit-V1.4.CHplugin" download>Windows Version</SpanA> <br /> <SpanA href="/plugins/Longer3d-Orange30-macOS-V1.4.CHplugin" download>Mac Version</SpanA> |
| Longer 3D Orange4K | <SpanA href="/plugins/Longer3D-Orange4K-Windows64bit-V1.5.CHplugin" download>Windows Version</SpanA> <br /> <SpanA href="/plugins/Longer3D-Orange4K-macOS-V1.5.CHplugin" download>Mac Version</SpanA> |

Once the download is complete, you can open the plugin window by going to **Top left drop-down menu > Help > Plugins**, and then import the plugin file.

## macOS

CHITUBOX Basic for macOS doesn't require an installation process since it is a portable application. You can simply drag the CHITUBOX icon to the Applications folder to install it.
