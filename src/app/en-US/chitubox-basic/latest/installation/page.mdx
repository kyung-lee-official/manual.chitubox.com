## macOS

CHITUBOX Basic for macOS doesn't require an installation process since it is a portable application. You can simply drag the CHITUBOX icon to the Applications folder to install it.

## Ubuntu

CHITUBOX Basic for Ubuntu doesn't require an installation process since it is a portable application. You can extract the downloaded package and run the CHITUBOX executable file to start the application.
