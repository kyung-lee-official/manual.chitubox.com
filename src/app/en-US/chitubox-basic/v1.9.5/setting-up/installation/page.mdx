# Installation

## Windows

Double click CHITUBOX installer to start installing.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation001.png"
	}
/>

Click **Next**.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation002--1.png"
	}
/>

Agree the EULA.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation003--1.png"
	}
/>

Select your path for installation, and click **Install**.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation004--1.png"
	}
/>

Wait for the progress bar until it is done.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation005--1.png"
	}
/>

Click **Yes**, restart the resource manager to continue.

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation006--1.png"
	}
/>

Click **Finish**, you're done!

<ImageContainer
	src={
		"/images/docs/en-US/chitubox-basic/1.x.x/004-setting-up-installation007--1.png"
	}
/>
